# Sleep function

The SLEEP function is used to introduce a delay or pause in the execution of a query or a batch of queries for a specific amount of time.

## `sleep`

The `sleep` function delays or pauses the execution of a query or a set of statements.



```surql title="API DEFINITION"
sleep(duration) -> none
```
The following example shows this function, and its output, when used in a [`RETURN`](/docs/surrealdb/1.1.x/surrealql/statements/return) statement:

```surql
RETURN sleep(1s);
RETURN sleep(500ms);
```
