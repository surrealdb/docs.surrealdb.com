## `parse::email::host`

The `parse::email::host` function parses and returns and email host from a valid email address.

```surql title="API DEFINITION"
parse::email::host(string) -> string
```
The following example shows this function, and its output, when used in a [`RETURN`](/docs/surrealdb/surrealql/statements/return) statement:

```surql
RETURN parse::email::host("info@surrealdb.com");

"surrealdb.com"
```
