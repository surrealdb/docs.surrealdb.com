# `BREAK` statement

<br /><br />

The BREAK statement can be used to skip an iteration of a loop, like within the [FOR-statement](/docs/surrealql/statements/for).


### Statement syntax

```surql title="SurrealQL Syntax"
BREAK
```

## Example usage

The following query shows example usage of this statement.

```surql
-- Create a person for everyone in the array
FOR $num IN [1, 2, 3, 4, 5, 6, 7, 8, 9] {
	IF ($num > 5) {
		BREAK;
	};
};
```
