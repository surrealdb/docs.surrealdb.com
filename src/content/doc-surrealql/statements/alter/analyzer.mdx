# `ALTER ANALYZER` statement

<Since v="v3.0.5" />

The `ALTER ANALYZER` statement can be used to modify an existing defined [analyzer](/docs/surrealql/statements/define/analyzer).

## Statement syntax

<Tabs syncKey="alter-statement">
  <TabItem label="SurrealQL Syntax">

```syntax title="SurrealQL Syntax"
ALTER ANALYZER [ IF EXISTS ] @name
  [ FUNCTION fn::@function | DROP FUNCTION ]
  [ TOKENIZERS @tokenizer, ... | DROP TOKENIZERS ]
  [ FILTERS @filter, ... | DROP FILTERS ]
  [ COMMENT @string | DROP COMMENT ]
```
