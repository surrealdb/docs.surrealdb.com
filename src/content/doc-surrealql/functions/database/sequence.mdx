## `sequence::next`

The `sequence::next` function returns the next value in a sequence.

```surql title="API DEFINITION"
sequence::next(sequence_name) -> int
```

```surql 
DEFINE SEQUENCE mySeq2 BATCH 1000 START 100 TIMEOUT 5s;
sequence::nextval('mySeq2');

-- 100
```
