# `->upsert()` {#upsert}

Creates or updates a record in a table.

```php title="Method Syntax"
$db->upsert($thing, $data);
```

### Example

```php title="Example"

$id = new RecordId('users', 'tobie');

// Upsert a record to the "users" collection.
$response = $db->upsert($id, [
	'firstname' => 'Tobie',
	'lastname' => 'Hitchcock',
]);
```
