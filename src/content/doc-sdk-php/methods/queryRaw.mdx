# `->queryRaw()` {#queryRaw}

Executes a query with a native response format from the targeted database.

```php title="Method Syntax"
$db->queryRaw($query, $params);
```

### Example

```php title="Example"

// Query the database for all users with the firstname "Tobie".
$response = $db->queryRaw(
	'SELECT * FROM users WHERE firstname = $firstname',
	['firstname' => 'Tobie']
);

// Output the response
var_dump($response);
```
