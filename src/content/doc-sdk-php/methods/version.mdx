# `->use()` {#use}

This method retrieves the current version of a remote database.

```php title="Method Syntax"
$db->version();
```

### Example usage
```php
$version = $db->version();
echo "The remote database is running version $version.";
```
```
The remote database is running version v1.5.4.
```
