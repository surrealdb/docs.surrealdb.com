# Feature table

This table summarises high-level differences. Exact availability can depend on your licence tier and release version—confirm details with your account team.

| Capability | Community Edition | Enterprise Edition |
| --- | --- | --- |
| Core SurrealQL and APIs | Yes | Yes |
| Single-node live queries | Yes | Yes |
| **Distributed live queries** (cluster-wide) | No | Yes |
| **Audit logging** (security-relevant events) | No | Yes |
| **FIPS 140-2** validated cryptography mode | No | Yes |
| **Trusted execution** (TEE) integration | No | Yes |
| **File / blob storage** (e.g. S3-compatible backends, `DEFINE BUCKET`) | No | Yes |
| Community support channels | Yes | Yes |
| **Commercial SLAs** and prioritised support | No | Yes (by tier) |

For installation and licence handling, see [Installation](/docs/manage/enterprise). For sales and contractual terms, see [Contact & licensing](/docs/manage/enterprise/overview/contact-and-licensing).
