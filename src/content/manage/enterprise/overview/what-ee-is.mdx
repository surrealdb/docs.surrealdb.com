# What EE is

**SurrealDB Enterprise Edition (EE)** builds on the open-source **Community Edition** with additional security, compliance, clustering, storage, and support options intended for production systems with strict operational requirements.

Beyond core SurrealQL and storage engines, EE focuses on **enterprise-grade capabilities**: hardened cryptography options, audit trails for governance, integrations that fit regulated industries, and features that scale live workloads across cluster nodes.

Typical **target use cases** include financial services, healthcare, government, and SaaS platforms that need contractual SLAs, security attestations, and predictable vendor assistance. Teams that already run SurrealDB in development on Community Edition can adopt EE without redesigning their data model—EE extends the platform rather than replacing it.
