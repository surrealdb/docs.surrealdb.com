The SurrealDB SDK for Go enables you to interact with SurrealDB from server-side applications, systems, and APIs. You can use the SDK to [execute queries](/docs/sdk/golang/concepts/executing-queries), [manage data](/docs/sdk/golang/concepts/data-manipulation), [authenticate users](/docs/sdk/golang/concepts/authentication), subscribe to real-time changes with [live queries](/docs/sdk/golang/concepts/live-queries), and work with interactive [sessions](/docs/sdk/golang/concepts/multiple-sessions) and [transactions](/docs/sdk/golang/concepts/transactions).

The SDK requires Go `1.23` or greater, is available as a [go.dev package](https://pkg.go.dev/github.com/surrealdb/surrealdb.go), and is compatible with SurrealDB `v2.x` and `v3.x`.
