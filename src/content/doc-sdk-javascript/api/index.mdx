# API Reference

This section provides comprehensive technical documentation for all classes, methods, types, and utilities in the SurrealDB JavaScript SDK. The API reference is organized by functionality to help you quickly find what you need.

## Core Classes

The core classes provide the foundation for connecting to SurrealDB and managing database sessions.

- [**Surreal**](/docs/sdk/javascript/api/core/surreal) - Main entry point for connecting to SurrealDB instances
- [**SurrealSession**](/docs/sdk/javascript/api/core/surreal-session) - Scoped session context with authentication and configuration
- [**SurrealQueryable**](/docs/sdk/javascript/api/core/surreal-queryable) - Base class providing query execution methods
- [**SurrealTransaction**](/docs/sdk/javascript/api/core/surreal-transaction) - Transaction management for atomic operations
- [**SurrealApi**](/docs/sdk/javascript/api/core/surreal-api) - User-defined API endpoint methods

## Query Builders

Query builder classes are returned by query methods and provide chainable configuration options before execution. [View all query builders →](/docs/sdk/javascript/api/queries/)

### Core Queries
- [**Query**](/docs/sdk/javascript/api/queries/query) - Raw SurrealQL query execution
- [**SelectPromise**](/docs/sdk/javascript/api/queries/select-promise) - SELECT queries with filtering and pagination
- [**CreatePromise**](/docs/sdk/javascript/api/queries/create-promise) - CREATE operations for new records
- [**InsertPromise**](/docs/sdk/javascript/api/queries/insert-promise) - Bulk INSERT operations
- [**UpdatePromise**](/docs/sdk/javascript/api/queries/update-promise) - UPDATE operations with merge/replace
- [**UpsertPromise**](/docs/sdk/javascript/api/queries/upsert-promise) - UPSERT operations (insert or replace)
- [**DeletePromise**](/docs/sdk/javascript/api/queries/delete-promise) - DELETE operations

### Graph & Real-time
- [**RelatePromise**](/docs/sdk/javascript/api/queries/relate-promise) - RELATE operations for graph relationships
- [**LivePromise**](/docs/sdk/javascript/api/queries/live-promise) - Live query subscriptions

### Functions & APIs
- [**RunPromise**](/docs/sdk/javascript/api/queries/run-promise) - Execute SurrealDB functions and SurrealML models
- [**ApiPromise**](/docs/sdk/javascript/api/queries/api-promise) - User-defined API endpoint calls

## Quick Navigation

### By Use Case

**Getting Started:**
1. Create a [`Surreal`](/docs/sdk/javascript/api/core/surreal) instance
2. Call [`connect()`](/docs/sdk/javascript/api/core/surreal#connect) to establish connection
3. Use query methods like [`select()`](/docs/sdk/javascript/api/core/surreal-queryable#select), [`create()`](/docs/sdk/javascript/api/core/surreal-queryable#create), etc.

**Authentication:**
- [`signup()`](/docs/sdk/javascript/api/core/surreal-session#signup) - Sign up new users
- [`signin()`](/docs/sdk/javascript/api/core/surreal-session#signin) - Sign in with credentials
- [`authenticate()`](/docs/sdk/javascript/api/core/surreal-session#authenticate) - Authenticate with tokens

**Querying Data:**
- [`query()`](/docs/sdk/javascript/api/core/surreal-queryable#query) - Raw SurrealQL
- [`select()`](/docs/sdk/javascript/api/core/surreal-queryable#select) - Select records ([SelectPromise](/docs/sdk/javascript/api/queries/select-promise))
- [`create()`](/docs/sdk/javascript/api/core/surreal-queryable#create) - Create records ([CreatePromise](/docs/sdk/javascript/api/queries/create-promise))
- [`insert()`](/docs/sdk/javascript/api/core/surreal-queryable#insert) - Insert records
- [`update()`](/docs/sdk/javascript/api/core/surreal-queryable#update) - Update records
- [`delete()`](/docs/sdk/javascript/api/core/surreal-queryable#delete) - Delete records

**Real-time Features:**
- [`live()`](/docs/sdk/javascript/api/core/surreal-queryable#live) - Subscribe to live queries
- [Event subscriptions](/docs/sdk/javascript/api/core/surreal#events) - Listen to connection events

**Transactions:**
- [`beginTransaction()`](/docs/sdk/javascript/api/core/surreal-session#begintransaction) - Start transactions
- [`commit()`](/docs/sdk/javascript/api/core/surreal-transaction#commit) - Commit changes
- [`cancel()`](/docs/sdk/javascript/api/core/surreal-transaction#cancel) - Roll back changes
