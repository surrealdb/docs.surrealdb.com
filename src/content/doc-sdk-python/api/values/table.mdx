# `Table` {#table}

A `Table` wraps a table name string, providing a type-safe way to reference SurrealDB tables. It can be used anywhere a [`RecordIdType`](/docs/sdk/python/api/values/#recordidtype) is accepted.

```python title="Import"
from surrealdb import Table
```

---


### Examples

```python
table = Table("users")
print(table.table_name)  # "users"
```

## Usage {#usage}

`Table` is interchangeable with a plain `str` in most SDK methods. It is useful when you want to distinguish table references from arbitrary strings at the type level.

```python
from surrealdb import Surreal, Table

db = Surreal("ws://localhost:8000")
db.connect()
db.use("my_ns", "my_db")
db.signin({"username": "root", "password": "root"})

users = db.select(Table("users"))
```
