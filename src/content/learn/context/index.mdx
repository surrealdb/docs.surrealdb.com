# Agent memory context (Spectron)

Spectron is a persistent agent memory layer built on SurrealDB. It provides knowledge graphs, automatic entity extraction, GraphRAG support, temporal facts, and hybrid retrieval for accurate context recall. Spectron enables AI agents to maintain long-term memory with high accuracy by combining graph structure, vector similarity, and time-aware facts in a single system.

## Why Spectron?

Spectron was built to resolve a problem faced by users of AI agents in which the agents themselves end up being little more than chatbots. Without an awareness of use preferences or structured memory to understand the relationship between entities, agents are unable to accumulate context or coordinate with each other.

## Details

Spectron will be coming soon, along with the documentation for how to use it. In the meantime, please see [this page](https://surrealdb.com/platform/spectron) for a top-level overview.

Spectron gives your AI agents persistent, queryable memory powered by knowledge graphs, entity extraction, temporal facts, and hybrid retrieval - built directly into SurrealDB rather than bolted on top.

- Automatically extract memories and facts from conversations
- Hybrid retrieval combining graph traversal and vector similarity
- Temporal awareness with bi-temporal, append-only facts
- Multi-agent shared memory with full ACID transactions
