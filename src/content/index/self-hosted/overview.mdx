# Running SurrealDB

For quick experimentation without needing to install anything, you can use the [Surrealist sandbox](https://app.surrealdb.com/) online. To persist your data while still experimenting, click on the "Deploy to Cloud" button where you can create a free SurrealDB Cloud instance to hold your data.

For self-hosting, you can run SurrealDB in a number of ways:

- [Run a single-node, in-memory server](/docs/self-hosted/in-memory) with optional persistence and versioning (SurrealMX)
- [Run a single-node, on-disk server](/docs/self-hosted/file-backed) (RocksDB or SurrealKV)
- [Run a multi-node, scalable cluster](/docs/self-hosted/multi-node)
- [Run with Docker](/docs/self-hosted/docker)
