# Installation

If you are not ready to install anything yet, start with the [Surrealist Sandbox](/docs/self-hosted/sandbox) or [SurrealDB Cloud](/docs/self-hosted/cloud) as described in the [Running overview](/docs/self-hosted/overview).

One of the most popular ways to get started with SurrealDB is to install and run it as a standalone database service, allowing any number of clients to connect to it and interact with the data.

Whether you are installing SurrealDB on your local machine for development purposes, or spinning up a production database, this section will guide you through the installation process, ensuring that you have all the necessary dependencies and configurations in place to start using SurrealDB effectively.

The current stable version of SurrealDB is <a href='/releases'> <Version /></a>.

## SurrealDB Cloud

Running and maintaining your own database can be a time consuming task. That's why we offer [SurrealDB Cloud](/docs/manage/cloud), a fully managed database service that allows you to focus on building your applications without worrying about the underlying infrastructure.
