# Schema commands

The package provides Artisan commands to preview and apply [Surqlize schema](/docs/languages/php/orm/surqlize/schema-dsl) definitions for the models registered in `config/surqlize.php`.

> [!WARNING]
> This package depends on the **alpha** v2 SDK and the in-development Surqlize ORM. Its commands may change.

## Commands

| Command | Options | Behavior |
|---|---|---|
| `surqlize:schema-dump` | `--connection=` | Print the generated `DEFINE` statements without executing them. |
| `surqlize:schema-apply` | `--dump`, `--dry-run`, `--connection=` | Apply the schema, or preview it with `--dump` / `--dry-run`. |

## Previewing a schema

```bash
php artisan surqlize:schema-dump
php artisan surqlize:schema-apply --dry-run
```

## Applying a schema

```bash
php artisan surqlize:schema-apply
php artisan surqlize:schema-apply --connection=analytics
```

Models come from `config('surqlize.models')`. If the list is empty, the command warns and exits without changes. Schema classes implement `Surqlize\Model\SchemaContract` with `definitions()` and `rules()`.
