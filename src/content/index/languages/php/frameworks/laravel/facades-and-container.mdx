# Facades and container

The package registers two service providers and three facades.

> [!WARNING]
> This package depends on the **alpha** v2 SDK. Its API may change.

## Facades

| Facade | Resolves to | Use it for |
|---|---|---|
| `SurrealDB` | `SurrealDBManager` | Lifecycle helpers and named-connection queries: `client()`, `connect()`, `disconnect()`, `reconnect()`, `isConnected()`, `health()`, `version()`, `run()`, `query()`, `using()`, `fake()`. |
| `Surreal` | `SurrealDB\SDK\Surreal` | The raw [SDK client](/docs/languages/php/v2/api/core/surreal): `connect()`, `run()`, `query()`, `live()`, and the fluent builders. |
| `Surqlize` | `SurqlizeManager` | ORM helpers: `executor()`, `schemaDefinitions()`, `applySchema()`, `transaction()`. |

```php
use SurrealDB\Laravel\Facades\SurrealDB;
use SurrealDB\Laravel\Facades\Surreal;
use SurrealDB\Laravel\Facades\Surqlize;

SurrealDB::health();
Surreal::run('RETURN true;');
Surqlize::transaction(fn ($tx) => User::createQuery(['name' => 'beau'], executor: $tx)->execute());
```

## Container bindings

### SDK provider

The `SurrealDBServiceProvider` binds:

- `SurrealDB\SDK\Surreal` and the `surrealdb` alias for the raw SDK client.
- `SurrealDB\Laravel\SurrealDBManager` plus the `surrealdb.manager` and `surrealdb.connection` aliases for the lifecycle manager (which implements the SDK `QueryExecutor`).
- `SurrealDB\Laravel\Factory\SurrealFactory`, which builds a `Surreal` client from config.

### ORM provider

The `SurqlizeServiceProvider` binds:

- `SurrealDB\SDK\Contracts\QueryExecutor` to a deferred executor that lazily resolves `config('surqlize.executor')` (default `surrealdb.connection`).
- `Surqlize\Model\SchemaManager` for schema generation and application.
- `SurrealDB\Laravel\SurqlizeManager` and the `surqlize` alias.

On boot, the ORM provider registers Surqlize's global `ConnectionManager` with this executor, so model queries run through the Laravel-managed connection automatically.

## Resolving from the container

```php
use SurrealDB\SDK\Surreal;
use SurrealDB\Laravel\SurrealDBManager;

$client = app(Surreal::class);
$manager = app(SurrealDBManager::class);
```
