# Migrating from v1

The v2 SDK is a full rewrite, not an incremental upgrade. This page summarizes the main differences so you can plan a migration.

> [!WARNING]
> The v2 SDK is an early **alpha** (`v2.0.0-alpha.0`). Its API may change before a stable release. The [v1 SDK](/docs/languages/php/v1) remains the stable choice for production.

## At a glance

| Area | v1 | v2 |
|---|---|---|
| Namespace | `Surreal\Surreal` | `SurrealDB\SDK\Surreal` |
| PHP version | `8.2`+ | `8.4`+ |
| Connecting | `connect($url)` then `use([...])` | `connect($url, new ConnectOptions(...))` |
| Authentication | array credentials | typed credential classes (`RootAuth`, ...) |
| CRUD | imperative methods (`create($thing, $data)`) | fluent builders (`create($thing)->content(...)->execute()`) |
| Values | mixed native/custom classes | typed `SurrealDB\SDK\Types\*` value classes |
| Runtimes | synchronous only | synchronous, OpenSwoole, Amp/Revolt |

## Namespace and client

```php
// v1
use Surreal\Surreal;
$db = new Surreal();
$db->connect('ws://127.0.0.1:8000/rpc');
$db->use(['namespace' => 'test', 'database' => 'test']);

// v2
use SurrealDB\SDK\Surreal;
use SurrealDB\SDK\Connection\ConnectOptions;
$db = new Surreal();
$db->connect('ws://127.0.0.1:8000/rpc', new ConnectOptions(
    namespace: 'test',
    database: 'test',
));
```

## Authentication

```php
// v1
$db->signin(['username' => 'root', 'password' => 'secret']);

// v2
use SurrealDB\SDK\Auth\RootAuth;
$db->signin(new RootAuth('root', 'secret'));
```

The `use()` method now takes positional arguments: `$db->use('test', 'test')`.

## CRUD operations

v2 replaces the imperative CRUD methods with fluent builders that you finish with `->execute()`.

```php
// v1
$person = $db->create('person:tobie', ['name' => 'Tobie']);
$db->merge('person:tobie', ['visits' => 2]);
$db->delete('person:tobie');

// v2
use SurrealDB\SDK\Types\RecordId;
$person = RecordId::from('person', 'tobie');
$db->create($person)->content(['name' => 'Tobie'])->execute();
$db->update($person)->merge(['visits' => 2])->execute();
$db->delete($person)->execute();
```

## Raw queries

`query()` in v1 becomes `run()` for raw SurrealQL in v2 (and `query()` now takes a [`BoundQuery`](/docs/languages/php/v2/concepts/bound-queries)).

```php
// v1
$result = $db->query('SELECT * FROM person', []);

// v2
$result = $db->run('SELECT * FROM person');
```

## Value types

v2 introduces a consistent set of typed value classes under `SurrealDB\SDK\Types` (for example `RecordId`, `Table`, `Uuid`, `DateTime`, `Duration`, `Decimal`, `Geometry*`). See [Codecs and typed values](/docs/languages/php/v2/concepts/codecs-and-typed-values).
