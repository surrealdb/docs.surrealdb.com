# Quickstart

This guide walks you through connecting, authenticating, and performing basic operations with the v2 SDK.

> [!WARNING]
> The v2 SDK is an early **alpha** (`v2.0.0-alpha.0`). Its API may change before a stable release. For production use today, use the [stable v1 SDK](/docs/languages/php/v1/start).

## 1. Install the SDK

Follow the [installation guide](/docs/languages/php/v2/installation) to add the SDK to your project, then include the Composer autoloader and import the classes you need.

```php
require __DIR__ . '/vendor/autoload.php';

use SurrealDB\SDK\Surreal;
use SurrealDB\SDK\Auth\RootAuth;
use SurrealDB\SDK\Connection\ConnectOptions;
use SurrealDB\SDK\Types\RecordId;
use SurrealDB\SDK\Types\Table;
```

## 2. Connect to SurrealDB

Create a `Surreal` instance and connect. The SDK selects an engine from the URL scheme: `ws`/`wss` for WebSocket and `http`/`https` for HTTP. Remote URLs automatically get a `/rpc` suffix when one is not present. Pass connection options such as the namespace, database, and authentication through `ConnectOptions`.

```php
$db = new Surreal();

$db->connect('ws://localhost:8000/rpc', new ConnectOptions(
    namespace: 'test',
    database: 'test',
    authentication: new RootAuth('root', 'root'),
));
```

See [Connecting to SurrealDB](/docs/languages/php/v2/concepts/connecting-to-surrealdb) for the differences between the HTTP and WebSocket engines, and [Authentication](/docs/languages/php/v2/concepts/authentication) for other credential types.

## 3. Create and read records

The fluent builders return typed builder objects. Call `->execute()` to run the statement, or `->compile()` to inspect the generated [bound query](/docs/languages/php/v2/concepts/bound-queries).

```php
$person = RecordId::from('person', 'tobie');

$created = $db->create($person)
    ->content(['name' => 'Tobie', 'visits' => 1])
    ->execute();

$selected = $db->select($person)->execute();

$people = $db->select(new Table('person'))->execute();
```

Because `$person` is a single `RecordId`, the SDK emits `ONLY`, so `->execute()` returns a single record. Selecting a `Table` returns a list of records.

## 4. Update and delete

```php
$db->update($person)
    ->merge(['visits' => 2])
    ->execute();

$db->delete($person)->execute();
```

## 5. Run SurrealQL

Use `run()` for raw SurrealQL with bindings. It returns one result per statement.

```php
$result = $db->run('SELECT * FROM type::table($tb)', [
    'tb' => new Table('person'),
]);
```

See [Executing queries](/docs/languages/php/v2/concepts/executing-queries) and [Fluent query builders](/docs/languages/php/v2/concepts/fluent-query-builders) for more.

## 6. Close the connection

```php
$db->close();
```
