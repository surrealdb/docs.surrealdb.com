# PHP SDK v2 (alpha)

The v2 SurrealDB SDK for PHP is a full rewrite of the PHP client under the `SurrealDB\SDK` namespace. It introduces fluent query builders, strongly typed SurrealDB values, pluggable HTTP and WebSocket engines, selectable runtimes (synchronous, OpenSwoole, and Amp/Revolt), live queries, and a PSR-based extension surface.

> [!WARNING]
> The v2 SDK is an early **alpha** (`v2.0.0-alpha.0`). Its API may change before a stable release, and it is not yet recommended for production. For production use today, use the [stable v1 SDK](/docs/languages/php/v1). The v2 SDK requires PHP `8.4` or greater.

> [!NOTE]
> The v2 SDK targets SurrealDB `2.x` and newer. Some features, such as sessions, server-side transactions, and refresh tokens, require SurrealDB `3.0.0` or later, and live queries require a WebSocket connection.
