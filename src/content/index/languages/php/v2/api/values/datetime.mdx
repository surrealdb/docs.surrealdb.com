# `DateTime` {#datetime}

A SurrealQL `datetime` value with nanosecond precision. Because PHP's native `DateTimeImmutable` only stores microseconds, this type keeps a `(seconds, nanoseconds)` pair to round-trip full precision.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Creating {#creating}

```php title="Syntax"
new DateTime(int $seconds = 0, int $nanoseconds = 0)
DateTime::fromString(string $iso): DateTime
DateTime::fromDateTime(DateTimeInterface $dateTime): DateTime
DateTime::now(): DateTime
DateTime::epoch(): DateTime
```

```php title="Examples"
use SurrealDB\SDK\Types\DateTime;

$now = DateTime::now();
$parsed = DateTime::fromString('2025-01-01T00:00:00Z');
```

## Methods {#methods}

```php title="Syntax"
$dt->toIso(): string                       // RFC 3339 string in UTC
$dt->toDateTimeImmutable(): DateTimeImmutable
$dt->toCompact(): array                    // [seconds, nanoseconds]
```
