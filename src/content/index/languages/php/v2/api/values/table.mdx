# `Table` {#table}

A SurrealDB table reference such as `person`. Table names are SurrealQL identifiers, so they are escaped inline rather than bound as parameters.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Constructor {#constructor}

```php title="Syntax"
new Table(string $name)
```

```php title="Example"
use SurrealDB\SDK\Types\Table;

$table = new Table('person');

$people = $db->select($table)->execute();
```
