# `Uuid` {#uuid}

A SurrealQL `uuid` value, backed by Symfony UID. Supports parsing from the canonical string or raw binary, and generating v4 (random) and v7 (time-ordered) UUIDs.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Creating {#creating}

```php title="Syntax"
Uuid::from(self|AbstractUid|string $uuid): Uuid
Uuid::fromString(string $uuid): Uuid
Uuid::fromBytes(string $bytes): Uuid
Uuid::v4(): Uuid
Uuid::v7(): Uuid
```

```php title="Examples"
use SurrealDB\SDK\Types\Uuid;

$random = Uuid::v4();
$ordered = Uuid::v7();
$parsed = Uuid::fromString('09748193-048a-4bfb-b825-8528cf74fdc1');
```

## Methods {#methods}

```php title="Syntax"
(string) $uuid           // canonical RFC 4122 string
$uuid->toBytes(): string // raw 16-byte representation
```
