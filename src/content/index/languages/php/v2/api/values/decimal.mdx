# `Decimal` {#decimal}

A SurrealQL `decimal` value: an arbitrary-precision decimal number backed by `Brick\Math\BigDecimal`. SurrealDB transports decimals as strings to preserve precision.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Constructor {#constructor}

```php title="Syntax"
new Decimal(self|BigDecimal|string|int|float $value)
```

```php title="Example"
use SurrealDB\SDK\Types\Decimal;

$price = new Decimal('19.99');
$total = $price->multipliedBy(3);
```
