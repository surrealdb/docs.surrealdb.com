# `Geometry` {#geometry}

The geometry value types represent GeoJSON geometries. Each derivative extends the abstract `Geometry` base.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Types {#types}

## Example {#example}

```php
use SurrealDB\SDK\Types\GeometryPoint;
use SurrealDB\SDK\Types\GeometryLine;

$point = new GeometryPoint(12.0, 49.0); // longitude, latitude

$line = new GeometryLine([
    new GeometryPoint(1.0, 2.0),
    new GeometryPoint(3.0, 4.0),
]);

$geoJson = $point->toGeoJson();
```

## Methods {#methods}

```php title="Syntax"
$geometry->toGeoJson(): array
$geometry->is(Geometry $other): bool
```

`GeometryPoint` also exposes `coordinates(): array` and the `longitude` / `latitude` properties.
