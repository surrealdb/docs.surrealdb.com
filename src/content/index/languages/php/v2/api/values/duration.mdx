# `Duration` {#duration}

A SurrealQL `duration` value with nanosecond precision, rendered using SurrealQL's compact unit syntax (for example `1h30m`, `500ms`, `0ns`).

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Creating {#creating}

```php title="Syntax"
new Duration(int $seconds = 0, int $nanoseconds = 0)
Duration::fromString(string $input): Duration
Duration::seconds(int $value): Duration
Duration::minutes(int $value): Duration
Duration::hours(int $value): Duration
Duration::days(int $value): Duration
Duration::milliseconds(int $value): Duration
```

```php title="Examples"
use SurrealDB\SDK\Types\Duration;

$ttl = Duration::fromString('1h30m');
$short = Duration::milliseconds(500);
```

## Methods {#methods}

```php title="Syntax"
$duration->add(Duration $other): Duration
$duration->sub(Duration $other): Duration
$duration->mul(int $factor): Duration
$duration->div(int $divisor): Duration
$duration->totalNanoseconds(): int
(string) $duration   // compact syntax, e.g. "1h30m"
```
