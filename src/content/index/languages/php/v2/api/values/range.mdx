# `Range` {#range}

A SurrealQL `range` value: a bounded or unbounded interval delimited by inclusive (`BoundIncluded`) or exclusive (`BoundExcluded`) bounds. A `null` bound denotes an open end.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Constructor {#constructor}

```php title="Syntax"
new Range(
    BoundIncluded|BoundExcluded|null $begin,
    BoundIncluded|BoundExcluded|null $end,
)
```

```php title="Example"
use SurrealDB\SDK\Types\Range;
use SurrealDB\SDK\Types\BoundIncluded;
use SurrealDB\SDK\Types\BoundExcluded;

$range = new Range(new BoundIncluded(1), new BoundExcluded(10)); // 1..10
$open = new Range(new BoundIncluded(1), null);                   // 1..
```

The escaped form renders SurrealQL range syntax such as `1..=10`, `1>..10`, `..=10`, or `1..`.
