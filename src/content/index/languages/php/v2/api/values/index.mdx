# Data types

The v2 SDK represents SurrealDB-specific types with dedicated value classes in the `SurrealDB\SDK\Types` namespace. They extend a common `Value` base and round-trip with their SurrealDB type intact. See [Codecs and typed values](/docs/languages/php/v2/concepts/codecs-and-typed-values) for how they are serialized.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.
