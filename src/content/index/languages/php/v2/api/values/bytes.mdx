# `Bytes` {#bytes}

A SurrealQL `bytes` value wrapping a raw binary string. Over the JSON protocol the bytes are carried base64url-encoded.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Creating {#creating}

```php title="Syntax"
new Bytes(string $bytes)
Bytes::fromBase64(string $base64): Bytes
```

```php title="Example"
use SurrealDB\SDK\Types\Bytes;

$bytes = new Bytes(random_bytes(16));
$encoded = $bytes->toBase64();
```

## Methods {#methods}

```php title="Syntax"
$bytes->toBase64(): string   // base64url, no padding
(string) $bytes              // the raw bytes
```
