# `Set` {#set}

A SurrealQL `set` value: an array whose items are deduplicated.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Constructor {#constructor}

```php title="Syntax"
new Set(iterable $items = [])
```

```php title="Example"
use SurrealDB\SDK\Types\Set;

$tags = new Set(['php', 'surrealdb', 'php']); // deduplicated to two items
```
