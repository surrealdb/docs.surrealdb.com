# `RecordId` {#record-id}

A fully-qualified SurrealDB record identifier such as `person:tobie`, combining a table name with an id. The id may be a string, integer, UUID, array, or object.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Constructor {#constructor}

```php title="Syntax"
new RecordId(string $table, string|int|array|object $id)
RecordId::from(string $table, string|int|array|object $id): RecordId
```

```php title="Examples"
use SurrealDB\SDK\Types\RecordId;

$id = new RecordId('person', 'tobie');
$id = RecordId::from('person', 'tobie');
$composite = RecordId::from('temperature', ['London', 2025]);
```

## Related types {#related}

- `StringRecordId` holds a record id in raw string form, for example `new StringRecordId('person:tobie')`, letting the server parse it.
- `RecordIdRange` represents a range of record ids.
