# `DriverOptions` {#driver-options}

`DriverOptions` is driver-wide configuration and the dependency-injection seam. Every field has a sensible default so `new Surreal()` works out of the box; override any field to swap an implementation. Pass it to the [`Surreal` constructor](/docs/languages/php/v2/api/core/surreal#constructor).

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Constructor {#constructor}

```php title="Syntax"
new DriverOptions(
    ?Codec $codec = null,
    CodecEnum $format = CodecEnum::JSON,
    ?EventDispatcherInterface $events = null,
    ?LoggerInterface $logger = null,
    ?ClientInterface $httpClient = null,
    ?RequestFactoryInterface $requestFactory = null,
    ?StreamFactoryInterface $streamFactory = null,
    ?Scheduler $scheduler = null,
    ?array $engines = null,
    ?\Closure $webSocketClientFactory = null,
    ?\Closure $webSocketTransportFactory = null,
    ?\Closure $httpTransportFactory = null,
    array $middleware = [],
    int $pingInterval = 30,
)
```

```php title="Example"
use SurrealDB\SDK\Connection\DriverOptions;
use SurrealDB\SDK\Enum\CodecEnum;

$db = new SurrealDB\SDK\Surreal(new DriverOptions(
    format: CodecEnum::CBOR,
    pingInterval: 15,
));
```
