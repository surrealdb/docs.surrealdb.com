# `ConnectOptions` {#connect-options}

`ConnectOptions` configures a specific connection. Pass it as the second argument to [`Surreal::connect()`](/docs/languages/php/v2/api/core/surreal#connect).

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Constructor {#constructor}

```php title="Syntax"
new ConnectOptions(
    ?string $namespace = null,
    ?string $database = null,
    Credentials|Token|AuthProviderInterface|\Closure|string|null $authentication = null,
    bool $versionCheck = true,
    bool $invalidateOnExpiry = false,
    bool|array|ReconnectStrategyInterface $reconnect = true,
)
```

```php title="Example"
use SurrealDB\SDK\Connection\ConnectOptions;
use SurrealDB\SDK\Auth\RootAuth;

$options = new ConnectOptions(
    namespace: 'test',
    database: 'test',
    authentication: new RootAuth('root', 'root'),
);
```
