# `Endpoint` {#endpoint}

`Endpoint` is a parsed, normalized SurrealDB endpoint. Remote schemes (`ws`, `wss`, `http`, `https`) get a `/rpc` suffix appended when one is missing. You can pass an `Endpoint` to [`Surreal::connect()`](/docs/languages/php/v2/api/core/surreal#connect) instead of a URL string.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Parsing {#parse}

```php title="Syntax"
Endpoint::parse(string $url): Endpoint
```

```php title="Example"
use SurrealDB\SDK\Connection\Endpoint;

$endpoint = Endpoint::parse('http://127.0.0.1:8000');
// $endpoint->uri === 'http://127.0.0.1:8000/rpc'
```

## Methods {#methods}

```php title="Syntax"
$endpoint->httpScheme(): string       // ws -> http, wss -> https
$endpoint->httpUri(): string          // the endpoint as an HTTP(S) URL
$endpoint->basePath(): string         // the path with a trailing /rpc removed
$endpoint->withPath(string $path): Endpoint
```
