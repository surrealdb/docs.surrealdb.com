# `ConnectionController` {#connection-controller}

`ConnectionController` orchestrates a connection: engine selection, connection and session state, version and feature gating, authentication renewal, and lifecycle events. Access it through [`Surreal::connection()`](/docs/languages/php/v2/api/core/surreal#connection-accessor) for advanced operations not exposed on the top-level client.

> [!WARNING]
> The v2 SDK is an early **alpha**. The methods below are advanced API and may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

```php title="Access"
$connection = $db->connection();
```

## Transactions {#transactions}

Require a WebSocket connection and SurrealDB `3.0.0` or later. See [Transactions](/docs/languages/php/v2/concepts/transactions).

```php title="Syntax"
$connection->begin(?string $session = null): string
$connection->commit(string $txn, ?string $session = null): void
$connection->cancel(string $txn, ?string $session = null): void
```

## Querying with a transaction {#query}

```php title="Syntax"
$connection->query(BoundQuery $query, ?string $session = null, ?string $txn = null): iterable
```

## Sessions {#sessions}

Require SurrealDB `3.0.0` or later.

```php title="Syntax"
$connection->sessions(): array
$connection->createSession(?string $clone = null): string
$connection->destroySession(?string $session): void
```

## Import and export {#import-export}

```php title="Syntax"
$connection->importSql(string $data): void
$connection->exportSql(array $options = []): string
```

## Token management {#tokens}

```php title="Syntax"
$connection->refresh(Tokens $tokens, ?string $session = null): Tokens
$connection->revoke(Tokens $tokens, ?string $session = null): void
```
