# `Surreal` {#surreal}

The `Surreal` class is the primary interface for connecting to SurrealDB, managing the session, authenticating, and running queries. It implements the `QueryExecutor` contract used by the fluent query builders.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Constructor {#constructor}

```php title="Syntax"
new Surreal(?DriverOptions $options = null)
```

```php title="Examples"
use SurrealDB\SDK\Surreal;
use SurrealDB\SDK\Runtime\Runtime;

$db = new Surreal();                 // synchronous JSON driver
$db = new Surreal(Runtime::swoole()); // OpenSwoole runtime
```

## Connection methods {#connection}

### `connect()` {#connect}

Opens a connection. Accepts a URL string or an [`Endpoint`](/docs/languages/php/v2/api/connection/endpoint), plus optional [`ConnectOptions`](/docs/languages/php/v2/api/connection/connect-options).

```php title="Syntax"
$db->connect(string|Endpoint $url, ?ConnectOptions $options = null): void
```

### `close()` {#close}

Closes the active connection.

```php title="Syntax"
$db->close(): void
```

### `status()` / `isConnected()` {#status}

```php title="Syntax"
$db->status(): ConnectionStatus
$db->isConnected(): bool
```

### `version()` / `health()` {#version}

```php title="Syntax"
$db->version(): string
$db->health(): void
```

### `subscribe()` {#subscribe}

Subscribes to a high-level connection event (`connecting`, `connected`, `reconnecting`, `disconnected`, `error`, `auth`, `using`) and returns an unsubscribe closure.

```php title="Syntax"
$db->subscribe(string $event, callable $listener): \Closure
```

## Session methods {#session}

```php title="Syntax"
$db->use(?string $namespace, ?string $database = null): void
$db->let(string $name, mixed $value): void
$db->unset(string $name): void
```

## Authentication methods {#auth}

```php title="Syntax"
$db->signin(Credentials|array $auth): Tokens
$db->signup(Credentials|array $auth): Tokens
$db->authenticate(Token|string $token): void
$db->invalidate(): void
```

See [Authentication](/docs/languages/php/v2/concepts/authentication) and the [Credentials reference](/docs/languages/php/v2/api/auth/credentials).

## Query methods {#queries}

### `run()` {#run}

Executes raw SurrealQL with optional bindings, returning one result per statement.

```php title="Syntax"
$db->run(string $surql, array $bindings = []): array
```

### `query()` {#query}

Executes a [`BoundQuery`](/docs/languages/php/v2/api/queries/bound-query), returning one result per statement.

```php title="Syntax"
$db->query(BoundQuery $query): array
```

## Live queries {#live}

Subscribes to a live query by id. WebSocket only.

```php title="Syntax"
$db->live(string $queryUuid): iterable
```

See [Live queries](/docs/languages/php/v2/concepts/live-queries).

## Advanced connection access {#connection-accessor}

Returns the underlying [`ConnectionController`](/docs/languages/php/v2/api/core/connection-controller) for advanced operations such as sessions, transactions, and import/export.

```php title="Syntax"
$db->connection(): ConnectionController
```
