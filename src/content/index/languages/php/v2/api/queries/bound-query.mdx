# `BoundQuery` {#bound-query}

`BoundQuery` is a parameter-bound SurrealQL fragment: the query text plus the values bound to generated placeholders. It is produced by the fluent builders (via `compile()`) and consumed by [`Surreal::query()`](/docs/languages/php/v2/api/core/surreal#query).

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Constructor {#constructor}

```php title="Syntax"
new BoundQuery(string $query = '', array $bindings = [])
```

## Methods {#methods}

```php title="Syntax"
$query->bind(mixed $value): string             // bind a value, returns "$bind_N"
$query->append(self|string $sql, array $bindings = []): self
```

`append()` throws an `ExpressionException` if a binding key collides.

```php title="Example"
use SurrealDB\SDK\Query\BoundQuery;

$query = new BoundQuery('SELECT * FROM person WHERE age > $min', ['min' => 18]);
$result = $db->query($query);
```
