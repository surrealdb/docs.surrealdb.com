# `RelateQuery` {#relate-query}

The fluent `RELATE` builder, creating graph edges between records. Created via [`Surreal::relate()`](/docs/languages/php/v2/api/core/surreal#builders). Pass single record ids, or arrays of record ids to relate many at once (the edge must then be a `Table`).

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

        <tr>
            <td><code>version(string $version)</code></td>
            <td><code>VERSION</code> for versioned storage.</td>
        </tr>
    </tbody>

```php title="Example"
use SurrealDB\SDK\Types\RecordId;
use SurrealDB\SDK\Types\Table;

$edge = $db->relate(
    RecordId::from('person', 'tobie'),
    new Table('likes'),
    RecordId::from('article', 'surrealdb'),
)->content(['strength' => 10])->execute();
```
