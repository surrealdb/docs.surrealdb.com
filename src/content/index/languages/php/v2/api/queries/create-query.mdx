# `CreateQuery` {#create-query}

The fluent `CREATE` builder. Created via [`Surreal::create()`](/docs/languages/php/v2/api/core/surreal#builders).

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Modifiers {#modifiers}

<table>
    <thead>
        <tr>
            <th>Method</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td><code>content(array|object $data)</code></td>
            <td>Set the record contents (<code>CONTENT</code>).</td>
        </tr>
        <tr>
            <td><code>patch(array $patches)</code></td>
            <td>Apply a JSON patch (<code>PATCH</code>).</td>
        </tr>
        <tr>
            <td><code>output(Output $output)</code></td>
            <td><code>RETURN</code> clause. See the <a href="/docs/languages/php/v2/concepts/fluent-query-builders#output-modifier">Output</a> enum.</td>
        </tr>
        <tr>
            <td><code>timeout(string $duration)</code></td>
            <td><code>TIMEOUT</code> duration.</td>
        </tr>
        <tr>
            <td><code>version(string $version)</code></td>
            <td><code>VERSION</code> for versioned storage.</td>
        </tr>
    </tbody>
</table>

```php title="Example"
use SurrealDB\SDK\Types\RecordId;

$created = $db->create(RecordId::from('person', 'tobie'))
    ->content(['name' => 'Tobie'])
    ->execute();
```
