# `SelectQuery` {#select-query}

The fluent `SELECT` builder. Created via [`Surreal::select()`](/docs/languages/php/v2/api/core/surreal#builders).

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Modifiers {#modifiers}

<table>
    <thead>
        <tr>
            <th>Method</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td><code>fields(string ...$fields)</code></td>
            <td>Project only the given fields.</td>
        </tr>
        <tr>
            <td><code>value(string $field)</code></td>
            <td>Return a single field's value (<code>SELECT VALUE</code>).</td>
        </tr>
        <tr>
            <td><code>where(string|BoundQuery $cond)</code></td>
            <td>Filter records.</td>
        </tr>
        <tr>
            <td><code>start(int $start)</code></td>
            <td><code>START</code> offset.</td>
        </tr>
        <tr>
            <td><code>limit(int $limit)</code></td>
            <td><code>LIMIT</code> count.</td>
        </tr>
        <tr>
            <td><code>fetch(string ...$fields)</code></td>
            <td>Resolve record-link fields (<code>FETCH</code>).</td>
        </tr>
        <tr>
            <td><code>timeout(string $duration)</code></td>
            <td><code>TIMEOUT</code> duration, for example <code>"5s"</code>.</td>
        </tr>
        <tr>
            <td><code>version(string $version)</code></td>
            <td><code>VERSION</code> for versioned storage.</td>
        </tr>
    </tbody>
</table>

```php title="Example"
$people = $db->select(new SurrealDB\SDK\Types\Table('person'))
    ->fields('name', 'age')
    ->where('age > 18')
    ->limit(10)
    ->execute();
```

## Execution {#execution}

```php title="Syntax"
$query->execute(): mixed       // run, returns the statement result
$query->compile(): BoundQuery  // inspect the generated query
```
