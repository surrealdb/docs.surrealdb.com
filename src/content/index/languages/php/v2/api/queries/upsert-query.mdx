# `UpsertQuery` {#upsert-query}

The fluent `UPSERT` builder. Created via [`Surreal::upsert()`](/docs/languages/php/v2/api/core/surreal#builders). It mirrors [`UpdateQuery`](/docs/languages/php/v2/api/queries/update-query) but emits an `UPSERT` statement.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

```php title="Example"
use SurrealDB\SDK\Types\RecordId;

$db->upsert(RecordId::from('person', 'tobie'))
    ->content(['name' => 'Tobie'])
    ->execute();
```
