# `DeleteQuery` {#delete-query}

The fluent `DELETE` builder. Created via [`Surreal::delete()`](/docs/languages/php/v2/api/core/surreal#builders). It defaults to `RETURN BEFORE`, so deleted records are returned.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Modifiers {#modifiers}

            <td><code>VERSION</code> for versioned storage.</td>
        </tr>
    </tbody>
</table>

```php title="Example"
use SurrealDB\SDK\Types\RecordId;

$deleted = $db->delete(RecordId::from('person', 'tobie'))->execute();
```
