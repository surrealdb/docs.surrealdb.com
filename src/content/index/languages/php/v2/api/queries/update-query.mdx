# `UpdateQuery` {#update-query}

The fluent `UPDATE` builder. Created via [`Surreal::update()`](/docs/languages/php/v2/api/core/surreal#builders).

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Modifiers {#modifiers}

<table>
    <thead>
        <tr>
            <th>Method</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td><code>content(array|object $data)</code></td>
            <td>Replace the record contents (<code>CONTENT</code>).</td>
        </tr>
        <tr>
            <td><code>merge(array|object $data)</code></td>
            <td>Merge fields into the record (<code>MERGE</code>).</td>
        </tr>
        <tr>
            <td><code>replace(array|object $data)</code></td>
            <td>Replace the record (<code>REPLACE</code>).</td>
        </tr>
        <tr>
            <td><code>patch(array $patches)</code></td>
            <td>Apply a JSON patch (<code>PATCH</code>).</td>
        </tr>
        <tr>
            <td><code>where(string|BoundQuery $cond)</code></td>
            <td>Filter affected records.</td>
        </tr>
        <tr>
            <td><code>output(Output $output)</code></td>
            <td><code>RETURN</code> clause.</td>
        </tr>
        <tr>
            <td><code>timeout(string $duration)</code></td>
            <td><code>TIMEOUT</code> duration.</td>
        </tr>
    </tbody>
</table>

```php title="Example"
use SurrealDB\SDK\Types\RecordId;

$db->update(RecordId::from('person', 'tobie'))
    ->merge(['visits' => 2])
    ->execute();
```
