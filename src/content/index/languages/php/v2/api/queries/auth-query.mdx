# `AuthQuery` {#auth-query}

The fluent builder that selects the currently-authenticated record by reading the `$auth` parameter (`SELECT * FROM ONLY $auth`). Created via [`Surreal::auth()`](/docs/languages/php/v2/api/core/surreal#builders).

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

```php title="Example"
$me = $db->auth()->execute();
```
