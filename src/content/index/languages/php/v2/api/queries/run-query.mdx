# `RunQuery` {#run-query}

The fluent builder for invoking a SurrealQL or SurrealML function (`fn::*`, `ml::*`, or built-ins). Created via [`Surreal::call()`](/docs/languages/php/v2/api/core/surreal#builders). Arguments are bound as parameters; the name and optional version are validated and emitted inline.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Creating {#creating}

```php title="Syntax"
$db->call(string $name, ?string $version = null, array $args = []): RunQuery
```

```php title="Example"
$greeting = $db->call('fn::greet', args: ['Tobie'])->execute();

$recommendations = $db->call('ml::recommend', '1.0.0', [$userId])->execute();
```
