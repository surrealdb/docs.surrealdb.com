# `InsertQuery` {#insert-query}

The fluent `INSERT` builder. Created via [`Surreal::insert()`](/docs/languages/php/v2/api/core/surreal#builders). Pass records directly, or a target `Table` plus records.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Modifiers {#modifiers}

<table>
    <thead>
        <tr>
            <th>Method</th>
            <th>Description</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td><code>relation()</code></td>
            <td>Insert into a relation table (<code>INSERT RELATION</code>).</td>
        </tr>
        <tr>
            <td><code>ignore()</code></td>
            <td>Ignore records that already exist (<code>INSERT IGNORE</code>).</td>
        </tr>
        <tr>
            <td><code>output(Output $output)</code></td>
            <td><code>RETURN</code> clause.</td>
        </tr>
        <tr>
            <td><code>timeout(string $duration)</code></td>
            <td><code>TIMEOUT</code> duration.</td>
        </tr>
        <tr>
            <td><code>version(string $version)</code></td>
            <td><code>VERSION</code> for versioned storage.</td>
        </tr>
    </tbody>
</table>

```php title="Example"
use SurrealDB\SDK\Types\Table;

$db->insert(new Table('person'), [
    ['name' => 'Tobie'],
    ['name' => 'Jaime'],
])->execute();
```
