# `Tokens` {#tokens}

`Tokens` is an access token and optional refresh token pair. It is returned by [`signin()`](/docs/languages/php/v2/api/core/surreal#auth) and [`signup()`](/docs/languages/php/v2/api/core/surreal#auth).

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Constructor {#constructor}

```php title="Syntax"
new Tokens(?string $access = null, ?string $refresh = null)
```

```php title="Example"
$tokens = $db->signin(new SurrealDB\SDK\Auth\RootAuth('root', 'root'));

echo $tokens->access;
```

Refresh tokens can be exchanged with [`ConnectionController::refresh()`](/docs/languages/php/v2/api/core/connection-controller#tokens) on SurrealDB `3.0.0` and later.
