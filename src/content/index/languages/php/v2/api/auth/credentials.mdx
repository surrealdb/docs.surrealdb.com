# Credentials {#credentials}

The credential classes in the `SurrealDB\SDK\Auth` namespace implement the `Credentials` interface and can be passed to [`signin()`](/docs/languages/php/v2/api/core/surreal#auth), [`signup()`](/docs/languages/php/v2/api/core/surreal#auth), or `ConnectOptions::$authentication`.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## `RootAuth` {#root-auth}

```php title="Syntax"
new RootAuth(string $username, string $password)
```

## `NamespaceAuth` {#namespace-auth}

```php title="Syntax"
new NamespaceAuth(string $namespace, string $username, string $password)
```

## `DatabaseAuth` {#database-auth}

```php title="Syntax"
new DatabaseAuth(string $namespace, string $database, string $username, string $password)
```

## `RecordAccessAuth` {#record-access-auth}

Record access (scope) credentials with arbitrary signin/signup variables.

```php title="Syntax"
new RecordAccessAuth(
    string $namespace,
    string $database,
    string $access,
    array $variables = [],
)
```

## `BearerAuth` {#bearer-auth}

A pre-issued access key.

```php title="Syntax"
new BearerAuth(string $namespace, string $database, string $access, string $key)
```

## `Token` {#token}

A raw JWT used with [`authenticate()`](/docs/languages/php/v2/api/core/surreal#auth).

```php title="Syntax"
new Token(string $value)
```

## The `Credentials` interface {#interface}

Each credential class exposes `toArray()`, which produces the parameter object expected by the `signin` / `signup` RPC methods.

```php title="Syntax"
interface Credentials
{
    public function toArray(): array;
}
```
