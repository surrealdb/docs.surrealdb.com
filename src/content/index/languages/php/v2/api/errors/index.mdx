# Errors

All v2 SDK exceptions extend `SurrealDB\SDK\Exceptions\SurrealException`, so you can catch them broadly or by specific type. See [Error handling](/docs/languages/php/v2/concepts/error-handling) for usage patterns.

> [!WARNING]
> The v2 SDK is an early **alpha**. Its API may change before a stable release.

**Source:** [surrealdb.php](https://github.com/surrealdb/surrealdb.php)

## Base {#base}

### `SurrealException` {#surrealexception}

Base class for all SDK exceptions.

## Server-side {#server-side}

Errors reported by the server are raised as `ServerException` (or a subclass) carrying a structured `kind`, `message`, `code`, and `details`.
