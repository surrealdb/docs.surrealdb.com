# PHP SDK v1

The v1 SurrealDB SDK for PHP is the stable, currently released SDK published on [Packagist](https://packagist.org/packages/surrealdb/surrealdb.php). It enables you to interact with SurrealDB from server-side applications, systems, and APIs, with functionality for executing queries, managing data, running database functions, authenticating to the database, building user signup and authentication functionality, and subscribing to data changes with live queries.

> [!NOTE]
> The latest stable version of the SDK is <Version sdk="php" />. The v1 SDK requires PHP version `8.2` or greater and uses the `Surreal\Surreal` class.

> [!IMPORTANT]
> A new [v2 SDK](/docs/languages/php/v2) is being developed as an alpha. It is a full rewrite under the `SurrealDB\SDK` namespace with fluent query builders, typed values, and runtime presets. If you are starting a new project and want to preview the next generation of the SDK, see the [v2 alpha documentation](/docs/languages/php/v2). For production usage today, stay on v1.
