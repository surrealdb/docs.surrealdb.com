# Quickstart

The v1 SDK for SurrealDB makes it straightforward to connect to your instance and start querying data. This guide walks you through connecting, authenticating, and performing basic operations.

> [!NOTE]
> This quickstart documents the stable v1 SDK, which uses the `Surreal\Surreal` class and array-based APIs. If you want the new fluent query builders and typed values, see the [v2 alpha quickstart](/docs/languages/php/v2/start).

## 1. Install the SDK

Follow the [installation guide](/docs/languages/php/v1/installation) to install the SDK as a dependency in your project. Once installed, include the Composer autoloader and import the `Surreal` class.

```php
require __DIR__ . '/vendor/autoload.php';

use Surreal\Surreal;
```

## 2. Connect to SurrealDB

Create a new instance and connect to your database endpoint. The `connect` method accepts either an `http` or `ws` URL. After connecting, select the namespace and database you want to work with.

```php
$db = new Surreal();

$db->connect("ws://127.0.0.1:8000/rpc");

$db->use([
    "namespace" => "main",
    "database" => "main",
]);
```

## 3. Authenticate

Sign in to your database. The example below uses root credentials.

```php
$token = $db->signin([
    "username" => "root",
    "password" => "secret",
]);
```

See [Authentication](/docs/languages/php/v1/concepts/authentication) for namespace, database, scope, and access signin options.

## 4. Create and select records

Use the [`create`](/docs/languages/php/v1/methods/create) method to insert a record, then the [`select`](/docs/languages/php/v1/methods/select) method to read it back.

```php
$person = $db->create("person:tobie", [
    "name" => "Tobie",
    "lastname" => "Morgan Hitchcock",
    "age" => 30,
]);

$people = $db->select("person");
```

## 5. Run SurrealQL queries

For more advanced use cases, use the [`query`](/docs/languages/php/v1/methods/query) method to run [SurrealQL](/docs/reference/query-language) statements directly. Use the second argument to safely pass variables.

```php
$result = $db->query(
    'SELECT * FROM type::table($tb)',
    [ "tb" => "person" ]
);
```

## 6. Close the connection

Close the connection when you are done to release resources.

```php
$db->close();
```

## What's next?

You have learned how to install the SDK, connect to SurrealDB, authenticate, create records, and retrieve data. There is a lot more you can do with the SDK.
