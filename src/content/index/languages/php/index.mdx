# PHP SDK

The SurrealDB SDK for PHP enables you to interact with SurrealDB from server-side applications, systems, and APIs, allowing you to integrate SurrealDB into your website or application backend, and serve dynamic content to your users. You can use the PHP SDK to execute queries, manage data, run database functions, authenticate to the database, build user signup and authentication functionality, and subscribe to data changes with live queries.

There are two generations of the SDK, plus an ORM and a Laravel integration in the wider ecosystem.

> [!IMPORTANT]
> The **v1 SDK** is the stable, currently released package on Packagist and is recommended for production use today. The **v2 SDK** is an early **alpha** rewrite under the `SurrealDB\SDK` namespace with fluent query builders, typed values, multiple runtimes, and live queries. Its API may change before a stable release.
