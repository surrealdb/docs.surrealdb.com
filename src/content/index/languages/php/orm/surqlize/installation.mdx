# Installation

Surqlize is installed with Composer and depends on the [v2 PHP SDK](/docs/languages/php/v2).

> [!WARNING]
> Surqlize is in early development and depends on the **alpha** v2 SDK. Its API may change. It requires PHP `8.4` or greater.

## Requirements

- PHP `8.4` or greater
- [Composer](https://getcomposer.org/download/)
- The [v2 PHP SDK](/docs/languages/php/v2/installation) (`surrealdb/surrealdb.php`)

## Install the package

```bash
composer require surqlize/surqlize
```

Because Surqlize depends on the alpha SDK, you may need to allow alpha stability in your project:

```json
{
    "minimum-stability": "alpha",
    "prefer-stable": true
}
```

If the SDK is not yet available to your Composer setup, add a repository entry before installing:

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/surrealdb/surrealdb.php"
        }
    ]
}
```
