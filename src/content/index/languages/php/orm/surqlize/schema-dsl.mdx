# Schema DSL

Surqlize manages schema through schema contracts that return SurrealDB `DEFINE` statements, plus PHP validation rules. You can write raw statements, use the fluent DSL, or mix both.

> [!WARNING]
> Surqlize is in early development. Its API may change.

## Schema contracts

Schemas implement `Surqlize\Model\SchemaContract`. Link a schema to a model with `#[Schema(...)]`.

```php
use Surqlize\Model\SchemaContract;

final class UserSchema implements SchemaContract
{
    public function definitions(): array
    {
        return [
            'DEFINE TABLE user SCHEMAFULL;',
            'DEFINE FIELD name ON user TYPE string;',
            'DEFINE FIELD age ON user TYPE int;',
        ];
    }

    public function rules(): array
    {
        return [
            'name' => static fn (mixed $value): bool|string =>
                is_string($value) && $value !== '' ? true : 'Name is required.',
        ];
    }
}
```

`definitions()` returns schema statements. `rules()` returns PHP validation callbacks that run before persistence operations such as `create()` and `save()`, returning `true` or an error string.

## Fluent DSL

The fluent DSL generates `DEFINE` statements for tables, fields, analyzers, indexes, and assertions.

```php
use Surqlize\Schema\Schema;

$schema = Schema::table('article')->schemafull();

$schema->analyzer('english')
    ->tokenizers(['class'])
    ->filters(['lowercase']);

$schema->field('title')
    ->string()
    ->assert(fn ($value) => $value->required()->minLength(3));

$schema->field('email')
    ->string()
    ->assert(fn ($value) => $value->email())
    ->unique('idx_article_email');

$schema->field('embedding')->vector(3);

$schema->index('idx_article_embedding')
    ->fields(['embedding'])
    ->hnsw(3);

$definitions = $schema->definitions();
```

You can mix raw strings and DSL objects in the same schema contract:

```php
public function definitions(): array
{
    return [
        'DEFINE TABLE legacy;',
        Schema::table('article')
            ->schemafull()
            ->field('title')
            ->string()
            ->assert(fn ($value) => $value->minLength(3)),
    ];
}
```

## Applying schemas

Apply schemas from PHP with the `SchemaManager`:

```php
use Surqlize\Model\SchemaManager;

(new SchemaManager())->apply([
    User::class,
    Address::class,
], $surreal);
```

Or use the [CLI](/docs/languages/php/orm/surqlize/cli):

```bash
vendor/bin/surqlize schema:apply surqlize.config.php
```
