# Surqlize

Surqlize is a SurrealDB ORM for PHP. It provides attribute-driven models, typed query helpers, graph edge traversal, mutation builders, schema tooling, and code generation on top of the [v2 PHP SDK](/docs/languages/php/v2).

The core idea is simple: describe your SurrealDB tables with PHP models and attributes, then compose SurrealQL through typed PHP APIs instead of string concatenation.

> [!WARNING]
> Surqlize is in early development and depends on the **alpha** [v2 PHP SDK](/docs/languages/php/v2) (`surrealdb/surrealdb.php`). Its API may change. It requires PHP `8.4` or greater.

## Features

- Attribute-driven table and edge models with `#[Table]`, `#[Edge]`, `#[Id]`, `#[Schema]`, `#[Cast]`, `#[Search]`, `#[Vector]`, and `#[Geometry]`.
- AST-based query compilation for deterministic SurrealQL output.
- Bound execution through the SurrealDB PHP SDK.
- Typed field adapters for IDE-friendly `select()`, `where()`, `orderBy()`, `fetch()`, and projection callbacks.
- Model helpers for create, update, upsert, delete, find, count, exists, and refresh.
- Graph traversal fields and `RELATE` support.
- Schema management through raw `DEFINE` statements or a fluent schema DSL.
- CLI commands for field adapter generation, schema application, and memory reports.

## Relationship to the SDK and Laravel

Surqlize is a separate package from the [PHP SDK](/docs/languages/php/v2). It uses the SDK's `QueryExecutor` and value types to run queries, but adds the model and ORM layer. For Laravel applications, the [Laravel integration](/docs/languages/php/frameworks/laravel) wires Surqlize into the framework's service container.
