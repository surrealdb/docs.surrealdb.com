# Models and attributes

Surqlize models are plain PHP classes annotated with attributes. Table models extend `Surqlize\Model\Model`; edge models extend `Surqlize\Edge\Edge`.

> [!WARNING]
> Surqlize is in early development. Its API may change.

## Defining a model

```php
use Surqlize\Attributes\Cast;
use Surqlize\Attributes\Id;
use Surqlize\Attributes\Schema;
use Surqlize\Attributes\Table;
use Surqlize\Model\Model;
use SurrealDB\SDK\Types\RecordId;

#[Table('user')]
#[Schema(UserSchema::class)]
final class User extends Model
{
    #[Id]
    public RecordId $id;

    public string $name;

    public int $age;

    #[Cast(Address::class)]
    public ?Address $address = null;
}
```

## Attributes

| Attribute | Target | Purpose |
|---|---|---|
| `#[Table('user')]` | Model class | Maps a model to a SurrealDB table. |
| `#[Edge('has_address', in: User::class, out: Address::class)]` | Edge class | Maps an edge model to a relation table and endpoint models. |
| `#[Id]` | Model property | Marks the model's `RecordId` property. |
| `#[Schema(UserSchema::class)]` | Model or edge class | Links a model to a [schema contract](/docs/languages/php/orm/surqlize/schema-dsl). |
| `#[Cast(Address::class)]` | Model property | Hydrates a nested value as another model. |
| `#[Search]` | Model property | Marks a field for [search](/docs/languages/php/orm/surqlize/typed-queries) helper generation. |
| `#[Vector(dimension: 3)]` | Model property | Marks a vector field and its dimension. |
| `#[Geometry]` | Model property | Marks a geometry-compatible field. |

> [!NOTE]
> `#[Id]` is a marker. The record id value is the SDK [`RecordId`](/docs/languages/php/v2/api/values/record-id) type. There is no field-alias attribute in v1, so PHP property names must match SurrealDB field names. See [Limitations](/docs/languages/php/orm/surqlize/limitations).

## Model methods

The base class provides query entry points and data operations.

| Method | Purpose |
|---|---|
| `select()` | Start a `SELECT` with fields or a typed field callback. |
| `query()` | Start `SELECT *` for the model table. |
| `selectValue()` | Start a `SELECT VALUE` query. |
| `relate()` | Start a model-first [relation builder](/docs/languages/php/orm/surqlize/edges-and-relate). |
| `create()` / `createQuery()` | Create a record. |
| `upsert()` | Upsert a record by id. |
| `save()` | Create or update based on whether the model has a `RecordId`. |
| `delete()` | Delete the model's current record. |
| `all()` | Fetch all rows as models. |
| `find()` / `findOrFail()` | Find one model by id. |
| `count()` / `exists()` | Aggregate helpers. |
| `refresh()` | Reload the current model from the database. |
| `toArray()` | Serialize initialized model properties. |
