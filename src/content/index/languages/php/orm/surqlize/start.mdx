# Quickstart

This guide walks you through configuring the connection, defining a model, and running typed queries with Surqlize.

> [!WARNING]
> Surqlize is in early development and depends on the **alpha** [v2 SDK](/docs/languages/php/v2). Its API may change.

## 1. Configure the executor

Surqlize runs queries through an SDK [`Surreal`](/docs/languages/php/v2/api/core/surreal) client. Register it with the global `ConnectionManager` during application bootstrap.

```php
use Surqlize\Connection\ConnectionManager;
use SurrealDB\SDK\Surreal;
use SurrealDB\SDK\Connection\ConnectOptions;
use SurrealDB\SDK\Auth\RootAuth;

$surreal = new Surreal();
$surreal->connect('ws://localhost:8000/rpc', new ConnectOptions(
    namespace: 'test',
    database: 'test',
    authentication: new RootAuth('root', 'root'),
));

ConnectionManager::set($surreal);
```

## 2. Define a model

Extend `Surqlize\Model\Model` and annotate the class with `#[Table]`. Mark the id property with `#[Id]`.

```php
use Surqlize\Attributes\Id;
use Surqlize\Attributes\Table;
use Surqlize\Model\Model;
use SurrealDB\SDK\Types\RecordId;

#[Table('user')]
final class User extends Model
{
    #[Id]
    public RecordId $id;

    public string $name;

    public int $age;
}
```

## 3. Compile and run a query

Use a typed field callback to build a `SELECT`. Call `compile()` to inspect the SurrealQL, or `collectModels()` to execute and hydrate models.

```php
$sql = User::select(fn ($user) => [$user->name, $user->age])
    ->where(fn ($user) => $user->age->gte(18))
    ->orderBy(fn ($user) => $user->name->asc())
    ->compile();

// SELECT name, age FROM user WHERE age >= 18 ORDER BY name ASC

$users = User::select(fn ($user) => [$user->id, $user->name])
    ->where(fn ($user) => $user->age->gte(18))
    ->collectModels();
```

## 4. Create a record

```php
$user = User::create([
    'name' => 'beau',
    'age' => 27,
]);
```
