# Surqlize (ORM)

[Surqlize](https://github.com/surrealdb/surqlize.php) is an object-relational mapper for SurrealDB in PHP. You describe tables as PHP classes with attributes, then build SurrealQL through a typed query builder instead of writing strings. It is built on top of [version 2 of the SDK](/docs/languages/php/v2) and uses the SDK to execute queries.

> [!IMPORTANT]
> Surqlize is published as `0.0.1-alpha.2` and is in early development. It requires PHP `8.4` and depends on the alpha v2 SDK, so its API may change.

## Installation

Surqlize depends on the alpha SDK, so allow alpha releases when installing.

```bash
composer require surrealdb/surqlize
```

## Defining a model

A model extends `Surqlize\Model\Model` and uses attributes to describe its table and fields. The `#[Id]` property holds an SDK `RecordId`.

```php
use Surqlize\Attributes\Id;
use Surqlize\Attributes\Table;
use Surqlize\Model\Model;
use SurrealDB\SDK\Types\RecordId;

#[Table('user')]
final class User extends Model
{
    #[Id]
    public RecordId $id;

    public string $name;
    public int $age;
}
```

| Attribute | Purpose |
|-----------|---------|
| `#[Table('name')]` | Maps the model to a table |
| `#[Id]` | Marks the record ID property |
| `#[Cast(Model::class)]` | Hydrates a nested model |
| `#[Schema(Schema::class)]` | Links a schema definition |
| `#[Edge('name', in: ..., out: ...)]` | Defines a graph edge model |

## Connecting

Surqlize runs queries through an SDK instance registered with the `ConnectionManager`.

```php
use Surqlize\Connection\ConnectionManager;
use SurrealDB\SDK\Surreal;

$db = new Surreal();
$db->connect(/* ... */);

ConnectionManager::set($db);
```

## Querying

The query builder takes closures that receive a typed field set, so conditions and projections are checked against your model.

```php
$users = User::select(fn ($user) => [$user->name, $user->age])
    ->where(fn ($user) => $user->age->gte(18))
    ->orderBy(fn ($user) => $user->name->asc())
    ->collectModels();

$user = User::find('tobie');
$count = User::count();
```

Mutations follow the same pattern.

```php
$user = User::create(['name' => 'Tobie', 'age' => 32], id: 'tobie');

$user->age = 33;
$user->save();

$user->delete();
```

## Graph relations

Edge models describe relationships, and `relate()` connects two records.

```php
User::relate($user)
    ->edge(HasAddress::class)
    ->with($address)
    ->content(['since' => 2024])
    ->execute();
```

## Schema and tooling

Surqlize can apply `DEFINE` statements from a `SchemaContract` or a fluent schema builder, and ships a CLI for code generation and schema application.

```bash
vendor/bin/surqlize generate:fields
vendor/bin/surqlize schema:apply
```
