# Golang SDK

The SurrealDB SDK for Go enables you to interact with SurrealDB from server-side applications, systems, APIs, and [as an embedded instance](/docs/languages/golang/embedding). You can use the SDK to [execute queries](/docs/languages/golang/concepts/executing-queries), [manage data](/docs/languages/golang/concepts/data-manipulation), [authenticate users](/docs/languages/golang/concepts/authentication), subscribe to real-time changes with [live queries](/docs/languages/golang/concepts/live-queries), and work with interactive [sessions](/docs/languages/golang/concepts/multiple-sessions) and [transactions](/docs/languages/golang/concepts/transactions).

The SDK requires Go `1.23` or greater, is available as a [go.dev package](https://pkg.go.dev/github.com/surrealdb/surrealdb.go), and is compatible with SurrealDB `v2.x` and `v3.x`.
