# `health()`

Runs a health check to verify that the server is reachable and accepting commands.

```python title="Method Syntax"
client.health()
```

### Arguments

This method takes no arguments.

### Example usage

```python
client.health()
```

If the server is unreachable, the call raises a `ConnectionError`.
