# Quickstart

The Kotlin SDK for SurrealDB makes it straightforward to connect to your instance and start querying data with coroutines. This guide walks you through connecting, authenticating, and performing basic operations.

> [!NOTE]
> Every network operation on the SDK is a `suspend` function, so the examples below run inside a coroutine (for example, a [`runBlocking`](https://kotlinlang.org/api/kotlinx.coroutines/kotlinx-coroutines-core/kotlinx.coroutines/run-blocking.html) block or a `CoroutineScope`).

## 1. Install the SDK

Follow the [installation guide](/docs/languages/kotlin/installation) to add the SDK as a dependency in your project. Once installed, import the client to start using it.

```kotlin
import com.surrealdb.kotlin.SurrealClient
import com.surrealdb.kotlin.SurrealClientConfig
```

## 2. Connect to SurrealDB

Create a [`SurrealClient`](/docs/languages/kotlin/api/core/surreal-client) with a [`SurrealClientConfig`](/docs/languages/kotlin/api/core/client-config), then call [`.use()`](/docs/languages/kotlin/api/core/surreal-client#use) to select a namespace and database, and [`.signin()`](/docs/languages/kotlin/api/core/surreal-client#signin) to authenticate.

The transport is selected automatically from the URL scheme:
- **WebSocket** (`ws://`, `wss://`) for long-lived stateful connections that support live queries, transactions, and multiple sessions
- **HTTP** (`http://`, `https://`) for short-lived stateless connections

```kotlin
import com.surrealdb.kotlin.SurrealClient
import com.surrealdb.kotlin.SurrealClientConfig
import kotlinx.coroutines.runBlocking
import kotlinx.serialization.json.buildJsonObject
import kotlinx.serialization.json.put

fun main() = runBlocking {
    val client = SurrealClient(SurrealClientConfig(url = "ws://localhost:8000"))

    client.signin(buildJsonObject {
        put("user", "root")
        put("pass", "root")
    })
    client.use("surrealdb", "docs")

    // ...

    client.close()
}
```

The client connects automatically on first use (`autoConnect` defaults to `true`). Call [`.close()`](/docs/languages/kotlin/api/core/surreal-client#close) when you are finished to release resources.

## 3. Inserting data

To represent records in your application, define [`@Serializable`](/docs/languages/kotlin/concepts/serialization) data classes that match your table structure.

```kotlin
import kotlinx.serialization.Serializable

@Serializable
data class Person(val name: String, val age: Int)
```

Use the [`.create()`](/docs/languages/kotlin/api/core/surreal-client#create) builder to insert a record. Pass a [`Table`](/docs/languages/kotlin/api/values/table) to create in a table with a generated ID, or a [`RecordId`](/docs/languages/kotlin/api/values/record-id) to create a record with a specific ID. Builders are terminated with `await()` (raw JSON) or the typed [`awaitAs<T>()`](/docs/languages/kotlin/api/core/query-builder#await-as) extension.

```kotlin
import com.surrealdb.kotlin.query.RecordId
import com.surrealdb.kotlin.query.Table
import com.surrealdb.kotlin.query.awaitAs
import kotlinx.serialization.json.buildJsonObject
import kotlinx.serialization.json.put

val created: Person = client
    .create(RecordId("person", "john"))
    .content(buildJsonObject {
        put("name", "John")
        put("age", 32)
    })
    .awaitAs()
```

## 4. Retrieving data

### Selecting records

The [`.select()`](/docs/languages/kotlin/api/core/surreal-client#select) builder retrieves records from a table or a single record by its [`RecordId`](/docs/languages/kotlin/api/values/record-id). Refine it with [`.where()`](/docs/languages/kotlin/api/core/query-builder#where) using the [expression helpers](/docs/languages/kotlin/api/core/query-builder#expressions).

```kotlin
import com.surrealdb.kotlin.query.Table
import com.surrealdb.kotlin.query.field
import com.surrealdb.kotlin.query.gte
import com.surrealdb.kotlin.query.awaitAs

val adults: List<Person> = client
    .select(Table("person"))
    .where(field("age") gte 18)
    .limit(50)
    .awaitAs()
```

### Running SurrealQL queries

For more advanced use cases, use [`.query()`](/docs/languages/kotlin/api/core/surreal-client#query) to execute [SurrealQL](/docs/reference/query-language) statements with bound [parameters](/docs/reference/query-language/language-primitives/parameters), or [`.queryAs<T>()`](/docs/languages/kotlin/api/core/surreal-client#query-as) to decode the result directly.

```kotlin
import kotlinx.serialization.json.buildJsonObject
import kotlinx.serialization.json.put

val people: List<Person> = client.queryAs(
    "SELECT * FROM person WHERE age > \$min_age",
    buildJsonObject { put("min_age", 25) },
)
```

## 5. Closing the connection

Call [`.close()`](/docs/languages/kotlin/api/core/surreal-client#close) to release the connection and all associated resources.

```kotlin
client.close()
```
