# `SurrealSession` {#surreal-session}

`SurrealSession` is the base class of [`SurrealClient`](/docs/languages/kotlin/api/core/surreal-client) and the type returned by [`.newSession()`](/docs/languages/kotlin/api/core/surreal-client#new-session). It exposes the full querying, authentication, and CRUD API with state ŌĆö authentication, namespace, database, and parameters ŌĆö isolated to that session. See [Multiple sessions](/docs/languages/kotlin/concepts/multiple-sessions) for the concept.

**Source:** [surrealdb.kotlin](https://github.com/surrealdb/surrealdb.kotlin)

---

## Shared API

Because [`SurrealClient`](/docs/languages/kotlin/api/core/surreal-client) extends `SurrealSession`, every session ŌĆö root or child ŌĆö provides the same methods documented on the client:

- Connection: [`.use()`](/docs/languages/kotlin/api/core/surreal-client#use), [`.ping()`](/docs/languages/kotlin/api/core/surreal-client#ping), [`.version()`](/docs/languages/kotlin/api/core/surreal-client#version)
- Authentication: [`.signin()`](/docs/languages/kotlin/api/core/surreal-client#signin), [`.signup()`](/docs/languages/kotlin/api/core/surreal-client#signup), [`.authenticate()`](/docs/languages/kotlin/api/core/surreal-client#authenticate), [`.auth()`](/docs/languages/kotlin/api/core/surreal-client#auth), [`.invalidate()`](/docs/languages/kotlin/api/core/surreal-client#invalidate), [`.reset()`](/docs/languages/kotlin/api/core/surreal-client#reset)
- Queries: [`.query()`](/docs/languages/kotlin/api/core/surreal-client#query), [`.queryAs<T>()`](/docs/languages/kotlin/api/core/surreal-client#query-as), [`.let()`](/docs/languages/kotlin/api/core/surreal-client#let), [`.unset()`](/docs/languages/kotlin/api/core/surreal-client#unset)
- CRUD builders: [`.select()`](/docs/languages/kotlin/api/core/surreal-client#select), [`.create()`](/docs/languages/kotlin/api/core/surreal-client#create), [`.update()`](/docs/languages/kotlin/api/core/surreal-client#update), [`.upsert()`](/docs/languages/kotlin/api/core/surreal-client#upsert), [`.merge()`](/docs/languages/kotlin/api/core/surreal-client#merge), [`.patch()`](/docs/languages/kotlin/api/core/surreal-client#patch), [`.delete()`](/docs/languages/kotlin/api/core/surreal-client#delete), [`.relate()`](/docs/languages/kotlin/api/core/surreal-client#relate), [`.insert()`](/docs/languages/kotlin/api/core/surreal-client#insert)
- Live queries: [`.live()`](/docs/languages/kotlin/api/core/surreal-client#live), [`.kill()`](/docs/languages/kotlin/api/core/surreal-client#kill)

## Creating and closing sessions

```kotlin title="Example"
val session = client.newSession()
session.signin(buildJsonObject { put("user", "a"); put("pass", "a") })
session.use("acme", "main")

// ... use the session ...

client.closeSession(session)
```

## Transactions

Transactions are started on a session via the [`transaction { }`](/docs/languages/kotlin/api/core/transaction#transaction-block) and [`beginTransaction()`](/docs/languages/kotlin/api/core/transaction#begin-transaction) extension functions.
