# Data manipulation

The Kotlin SDK provides fluent builders for the common CRUD operations. Each builder method (such as [`.select()`](/docs/languages/kotlin/api/core/surreal-client#select) or [`.create()`](/docs/languages/kotlin/api/core/surreal-client#create)) returns a [query builder](/docs/languages/kotlin/api/core/query-builder) that you refine and then terminate with `await()` (raw [`JsonElement`](/docs/languages/kotlin/concepts/value-types)) or the typed [`awaitAs<T>()`](/docs/languages/kotlin/api/core/query-builder#await-as) extension. Under the hood these compile to [SurrealQL](/docs/reference/query-language) and dispatch through [`.query()`](/docs/languages/kotlin/concepts/executing-queries), mirroring the [JavaScript SDK](/docs/languages/javascript/start).

The `what` argument accepts a [`Table`](/docs/languages/kotlin/api/values/table) to target every record in a table, a [`RecordId`](/docs/languages/kotlin/api/values/record-id) to target a single record, or a [`RecordIdRange`](/docs/languages/kotlin/api/values/record-id-range) to target a range.

## Creating records

Build content with [`buildJsonObject`](/docs/languages/kotlin/concepts/value-types) and finish with the typed [`awaitAs<T>()`](/docs/languages/kotlin/api/core/query-builder#await-as).

```kotlin
import com.surrealdb.kotlin.query.RecordId
import com.surrealdb.kotlin.query.awaitAs
import kotlinx.serialization.Serializable
import kotlinx.serialization.json.buildJsonObject
import kotlinx.serialization.json.put

@Serializable
data class Person(val name: String, val age: Int)

val ada: Person = client
    .create(RecordId("person", "ada"))
    .content(buildJsonObject {
        put("name", "Ada")
        put("age", 36)
    })
    .awaitAs()
```

## Selecting records

Refine a [`select`](/docs/languages/kotlin/api/core/query-builder#select-query) with [`.where()`](/docs/languages/kotlin/api/core/query-builder#where), [`.limit()`](/docs/languages/kotlin/api/core/query-builder#limit), [`.start()`](/docs/languages/kotlin/api/core/query-builder#start), [`.fetch()`](/docs/languages/kotlin/api/core/query-builder#fetch), and others, using the [expression helpers](/docs/languages/kotlin/api/core/query-builder#expressions).

```kotlin
import com.surrealdb.kotlin.query.Table
import com.surrealdb.kotlin.query.field
import com.surrealdb.kotlin.query.gte
import com.surrealdb.kotlin.query.awaitAs

val adults: List<Person> = client
    .select(Table("person"))
    .where(field("age") gte 18)
    .limit(50)
    .awaitAs()
```

## Updating and merging

Use [`.update()`](/docs/languages/kotlin/api/core/surreal-client#update) to replace record content, [`.merge()`](/docs/languages/kotlin/api/core/surreal-client#merge) to merge data, or [`.upsert()`](/docs/languages/kotlin/api/core/surreal-client#upsert) to create or update. Control the returned payload with [`.returnMode()`](/docs/languages/kotlin/api/core/query-builder#return-mode).

```kotlin
import com.surrealdb.kotlin.query.RecordId
import com.surrealdb.kotlin.query.ReturnMode
import kotlinx.serialization.json.buildJsonObject
import kotlinx.serialization.json.put

client
    .merge(RecordId("person", "ada"), buildJsonObject { put("age", 37) })
    .returnMode(ReturnMode.After)
    .await()
```

## Deleting records

```kotlin
import com.surrealdb.kotlin.query.RecordId

client.delete(RecordId("person", "ada")).await()
```

## Relating records

Create a graph edge between two records with [`.relate()`](/docs/languages/kotlin/api/core/surreal-client#relate).

```kotlin
import com.surrealdb.kotlin.query.RecordId
import kotlinx.serialization.json.buildJsonObject
import kotlinx.serialization.json.put

client
    .relate(RecordId("person", "ada"), RecordId("wrote", "w1"), RecordId("article", "a1"))
    .content(buildJsonObject { put("year", 1843) })
    .await()
```
