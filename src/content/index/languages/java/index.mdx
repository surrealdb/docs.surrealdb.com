# Java SDK

The SurrealDB SDK for Java lets you connect to [SurrealDB](/docs/start) from any Java application. It supports connecting to remote instances over WebSocket or HTTP, and running embedded databases in-process. The SDK provides methods for querying with [SurrealQL](/docs/reference/query-language), managing data, [authentication](/docs/learn/security/authentication/authentication), live queries, and transactions. It uses JNI to call native Rust code for high performance.
