The JavaScript SDK translates all data types native to SurrealQL into either data types native to JavaScript, or custom implementations. This document describes all data types, and links to their respective documentation.

## `RecordId` {#recordid}

A `RecordId` is used to represent a record identifier in SurrealDB. It consists of a table name and a unique identifier.

```ts
import { RecordId } from 'surrealdb';

const rid = new RecordId('person', 'tobie');
```

The unique identifier can be a `string`, `number`, `bigint`, or even an `object` or `array`.

```ts
new RecordId('person', 'tobie');
new RecordId('temperature', new Date());
new RecordId('location', [51.509865, -0.118092]);
new RecordId('city', { country: 'UK', name: 'London' });
```

### `StringRecordId`

If you need to pass a record ID as a raw string (for example, when using complex record IDs), you can use the `StringRecordId` class.

```ts
import { StringRecordId } from 'surrealdb';

const rid = new StringRecordId('person:tobie');
```

> [!NOTE]
> The `StringRecordId` class is useful for advanced record ID formats that may not be easily represented with the `RecordId` constructor.

<br />

## `Table` {#table}

A `Table` is used to represent a table name in SurrealDB. It is a type-safe wrapper around a table name string.

```ts
import { Table } from 'surrealdb';

const table = new Table('person');
```

<br />

## `Duration` {#duration}

A `Duration` is used to represent a time duration.

```ts
import { Duration } from 'surrealdb';

const dur = new Duration('1d2h30m');
```

<br />

## `Decimal` {#decimal}

A `Decimal` is used to represent an arbitrary precision decimal number.

```ts
import { Decimal } from 'surrealdb';

const dec = new Decimal('3.333333');
```

<br />

## `Uuid` {#uuid}

A `Uuid` is used to represent a universally unique identifier.

```ts
import { Uuid } from 'surrealdb';

const id = new Uuid('92b84bde-39c8-4b4b-92f7-626096d6c4d9');
```

<br />

## `Datetime` {#datetime}

Datetimes are represented as standard JavaScript `Date` objects.

```ts
const date = new Date('2024-05-06T17:44:57.085Z');
```

<br />

## Geometry types {#geometry}

The SDK provides custom classes for GeoJSON geometry types:

```ts
import {
    GeometryPoint,
    GeometryLine,
    GeometryPolygon,
    GeometryMultiPoint,
    GeometryMultiLine,
    GeometryMultiPolygon,
    GeometryCollection,
} from 'surrealdb';

const point = new GeometryPoint([51.509865, -0.118092]);
const line = new GeometryLine([
    new GeometryPoint([1, 2]),
    new GeometryPoint([3, 4]),
]);
```

<br />

## Converting to JSON

You can convert any response containing custom data types into a JSON-safe representation using the [`jsonify()`](/docs/1.x/sdk/javascript/core/utilities#jsonify) utility.

```ts
import { jsonify } from 'surrealdb';

const data = {
    rid: new RecordId('person', 'tobie'),
    dec: new Decimal('3.333333'),
    dur: new Duration('1d2h'),
    tb: new Table('table_name'),
};

const json = jsonify(data);
```
