# Go SDK

The SurrealDB SDK for Go enables you to interact with SurrealDB from server-side applications, systems, APIs, and [as an embedded instance](/docs/reference/golang/embedding). You can use the SDK to [execute queries](/docs/reference/golang/concepts/executing-queries), [manage data](/docs/reference/golang/concepts/data-manipulation), [authenticate users](/docs/reference/golang/concepts/authentication), subscribe to real-time changes with [live queries](/docs/reference/golang/concepts/live-queries), and work with interactive [sessions](/docs/reference/golang/concepts/multiple-sessions) and [transactions](/docs/reference/golang/concepts/transactions).

> [!IMPORTANT]
> The SDK requires Go `1.23` or greater, and is available as a [go.dev package](https://pkg.go.dev/github.com/surrealdb/surrealdb.go).

> [!NOTE]
> The latest version of the SDK is <Version sdk="golang" />.
> The SDK works with SurrealDB versions `v2.0.0` to <Version />, ensuring compatibility with the latest version.
