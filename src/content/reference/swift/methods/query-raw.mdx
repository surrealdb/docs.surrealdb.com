# `queryRaw()` {#query-raw}

Executes a raw SurrealQL string with bound parameters and returns one [`RPCQueryResult`](/docs/reference/swift/data-types#rpcqueryresult) per statement.

```swift title="Method Syntax"
try await client.queryRaw(sql, bindings: bindings)
```

### Example usage

```swift
let results: [RPCQueryResult] = try await client.queryRaw(
    "SELECT * FROM person WHERE age > $minAge LIMIT $limit;",
    bindings: [
        "minAge": .int(18),
        "limit": .int(50)
    ]
)

for row in results {
    if row.status == .ok {
        print(row.result)
    }
}
```
