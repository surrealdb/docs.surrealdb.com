# MongoDB

Create file users.json:

```json
[
  {
    "_id": { "$oid": "64a7a21fc25e3d2c7a3b55f1" },
    "name": "Alice Johnson",
    "email": "alice@example.com",
    "age": 29,
    "roles": ["admin", "editor"]
  },
  {
    "_id": { "$oid": "64a7a21fc25e3d2c7a3b55f2" },
    "name": "Bob Smith",
    "email": "bob@example.com",
    "age": 35,
    "roles": ["user"]
  }
]
```

Import data to MongoDB:

```
mongoimport --db testdb --collection users --file users.json --jsonArray
```

Start the MongoDB server

```
mongod --dbpath /your/current/directory
```

Import data to MongoDB:

```
surreal-sync sync mongo-db \
  --source-database "testdb" \
  --to-namespace "production" \
  --to-database "migrated_data" \
  --source-uri "mongodb://localhost:27017"
```

Log in to SurrealDB database under namespace `production` and database `migrated_data`, use `SELECT * FROM users` query:

```surql title="Result"
[
	{
		age: 29,
		email: 'alice@example.com',
		id: users:64a7a21fc25e3d2c7a3b55f1,
		name: 'Alice Johnson',
		roles: [
			'admin',
			'editor'
		]
	},
	{
		age: 35,
		email: 'bob@example.com',

```
