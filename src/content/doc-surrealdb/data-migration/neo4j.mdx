# Neo4J

Log in to Neo4J, start server with user `neo4j` and password `password`, then run the following queries:

```title="Sample Neo4J data"
CREATE (a:Person {name: 'Alice', age: 29})
CREATE (b:Person {name: 'Bob', age: 35})
CREATE (c:Person {name: 'Charlie', age: 40})
CREATE (a)-[:FRIENDS_WITH]->(b)
CREATE (b)-[:FRIENDS_WITH]->(c)
```

Run the following command to import the data to SurrealDB:

```
surreal-sync sync
neo4j \
  --source-uri "bolt://localhost:7687" \
  --source-username "neo4j" \
  --source-password "password" \
  --to-namespace "test" \
  --to-database "test"
```

Log in to SurrealDB under namespace `test` and database `test`, use `SELECT *, friends_with->person FROM person` statement to see imported data:
