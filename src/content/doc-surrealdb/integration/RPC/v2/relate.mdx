# `relate`

```json title="Method Syntax"
relate [
    in,
    relation,
    out,
    data?,
    {
        unique?: boolean,
        only?: boolean,
        output?: "none" | "null"| "diff" | "before" | "after",
        timeout?: duration,
        version:? datetime,
        vars?: object
    }?
]
```
