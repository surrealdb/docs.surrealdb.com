# `select`

```json title="Method Syntax"
select [ 
    thing,  
    {
        fields?: string,
        only?: boolean, 
        start?: number,
        limit?: number,
        cond?: string,
        timeout?: duration, 
        version?: datetime, 
        vars?: object
    }?
]
```
