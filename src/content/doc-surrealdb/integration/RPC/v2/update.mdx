# `update`

```json title="Method Syntax"
insert [ 
    thing, 
    data?, 
    {
        data_expr?: "content" | "merge" | "replace" | "patch",
        only?: boolean,
        cond?: string,
        output?: "none" | "null"| "diff" | "before" | "after",
        timeout?: duration, 
        vars?: object
    }? 
]
```
