# `create`

```json title="Method Syntax"
create [ 
    thing, 
    data?, 
    {
        only?: boolean, 
        output?: "none" | "null"| "diff" | "before" | "after",
        timeout?: duration, 
        version?: datetime, 
        vars?: object
    }?
]
```
