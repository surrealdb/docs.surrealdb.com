# `insert`

```text title="Method Syntax"
insert [ 
    thing, 
    data, 
    {
        data_expr?: "content" | "single",
        relation?: boolean, 
        output?: "none" | "null"| "diff" | "before" | "after",
        timeout?: duration, 
        version?: datetime, 
        vars?: object
    }?
]
```
