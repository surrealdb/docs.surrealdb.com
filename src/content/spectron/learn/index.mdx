Spectron is a persistent agent memory layer that gives AI agents long-term, queryable memory by combining knowledge graphs, automatic entity extraction, GraphRAG retrieval, and temporal facts in a single system.

This documentation is dedicated to Spectron and is the home for guides, references, and SDK documentation as the product evolves.

## What you can do with Spectron

- Maintain accurate, structured memory across multi-turn conversations and multi-agent workflows.
- Combine graph traversal with vector similarity for hybrid retrieval that prefers correctness over fuzzy guesses.
- Capture temporal facts so memories reflect when something was true, not just whether it was ever true.
- Inspect, edit, and govern what your agents remember through a clear API and a managed console.

## Getting started

Spectron is rolling out in stages. To follow along:

1. Read the [Spectron product page](https://surrealdb.com/platform/spectron) for a top-level overview of capabilities and use cases.
2. Sign in to [SurrealDB Cloud](https://app.surrealdb.com/signin) and create a Spectron context to obtain a base URL and API key.
3. Use the API key with your preferred SDK to start storing and retrieving memories.

More detailed guides, the full API reference, and SDK documentation will land in this section as Spectron progresses toward general availability.
