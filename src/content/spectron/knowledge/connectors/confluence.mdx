# Confluence connector

> **Not implemented.** No Confluence Cloud integration ships with the current release.

**Workaround:** export spaces to HTML or PDF and ingest via the [document upload](/docs/spectron/knowledge/ingestion/multimodal-content) pipeline.

See [Connectors overview](/docs/spectron/knowledge/connectors/overview).
