# Embedded library quickstart

Spectron runs as a **horizontally scalable service** in front of SurrealDB. Integrate through:

- **HTTP** — `/api/v1/{context_id}/...` ([REST API](/docs/spectron/integrations/surfaces/rest))
- **MCP** — `/mcp` on the same port
- **Generated SDKs** — Python (`surrealdb`) and TypeScript (`@surrealdb/spectron`)
- **Harness adapters** — LangChain, Vercel AI SDK, OpenAI Agents, n8n, Claude Code hook ([Integrations](/docs/spectron/integrations))

## In-process library

There is **no** supported in-process API that runs extraction and recall inside your application binary without the Spectron server. Rust, Python, and TypeScript agents call the HTTP API or SDK against a deployed Spectron instance.

## What to use

| Goal | Use |
| --- | --- |
| Python / TypeScript agent | [Python](/docs/spectron/integrations/sdks/python) or [JavaScript SDK](/docs/spectron/integrations/sdks/javascript-and-typescript) |

| Self-hosted stack | [Self-hosted quickstart](/docs/spectron/quickstarts/self-hosted) or [Docker](/docs/spectron/self-hosting/deployment/docker) |
