# Spectron documentation

Spectron is a **memory and knowledge layer for AI agents** – stateless application tier, **SurrealDB** as the multi-model store (graph, vector, document, relational, geospatial) with **ACID writes**, **first-class provenance and trust**, **graph-resident traces**, and **tri-temporal** belief history. The design goal is familiar: memory that **associates** related ideas and keeps straight **what was said, what is true now, and what used to be true** – much like people do, but queryable and auditable for software.

Use this hub to go from principles to running code, then dive into operational sections (knowledge ingestion, conversational memory, self-hosting, integrations, reference).
