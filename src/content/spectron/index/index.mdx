# Get started

This section takes you from zero to a working Spectron integration. Start with a quickstart to get code running, then use the mental model pages to understand what Spectron is doing under the hood.

## Quickstarts

The fastest path to a working session:

- **[Hosted quickstart](/spectron/get-started/quickstarts/hosted)** — connect to Spectron Cloud with an API key, run a session, and inspect stored memory. No infrastructure required.
- **[Self-hosted quickstart](/spectron/get-started/quickstarts/self-hosted)** — pull the Docker image, point it at a SurrealDB instance, and start the server locally.
