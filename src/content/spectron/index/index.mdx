# Spectron documentation

Spectron is a **memory and knowledge layer for AI agents** – stateless application tier, **SurrealDB** as the multi-model substrate (graph, vector, document, relational, geospatial) with **ACID writes**, **first-class provenance**, **graph-resident traces**, and **tri-temporal** belief history.

Use this hub to go from principles to running code, then dive into operational sections (knowledge ingestion, conversational memory, self-hosting, integrations, reference).
