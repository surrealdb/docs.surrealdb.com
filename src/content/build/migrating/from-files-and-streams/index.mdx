# Migrating from files and streams to SurrealDB

This section details how to map data, queries and concepts you may know from other databases and datatypes into SurrealDB.

The sources in this section of the documentation can be automatically imported to SurrealDB using the [Surreal Sync](https://github.com/surrealdb/surreal-sync/) tool. In addition, [CSV data](/docs/build/migrating/csv) can be automatically imported into Surrealist using its own [built-in functionality](/docs/explore/surrealist/concepts/exploring-database-records#importing-a-csv-file).

For other sources not yet supported, consider beginning by exporting the database as JSON which can be [imported on the command line](/docs/reference/cli/surrealdb-cli/import). Alternatively, you can use [one of the many available SDKs](/docs/start) to access your existing database and transfer its content directly to a SurrealDB instance.

* [JSON lines](/docs/build/migrating/jsonl)
