# Better Auth

[Better Auth](https://better-auth.com) is a comprehensive, framework-agnostic authentication and authorisation library for TypeScript. The `@surrealdb/better-auth` adapter lets you use SurrealDB as the database behind Better Auth: you pass it a single connected [`Surreal`](/docs/languages/javascript) client and Better Auth manages all of its table operations through the adapter.

The adapter supports schema generation, transactions, the full set of Better Auth WHERE operators, and every Better Auth plugin.

> [!NOTE]
> The `@surrealdb/better-auth` adapter works with SurrealDB versions `v3.1.0` and later.

## Prerequisites

- SurrealDB 3.1 or later
- Better Auth 1.6.x
- Node.js 18+ or Bun 1.x

## Feature support

The adapter advertises the following capabilities to Better Auth:

| Capability | Supported |
|------------|-----------|
| JSON / object fields | Yes |
| Dates | Yes |
| Booleans | Yes |
| Arrays | Yes |
| Numeric IDs | No |
| UUID IDs | No |

Records use Better Auth's default string identifiers, which the adapter stores as SurrealDB [record IDs](/docs/surrealql/datamodel/ids) and deserialises back to strings. Numeric and UUID identifier strategies are not supported. See [Transactions & limitations](/docs/build/integrations/authentication/transactions-and-limitations) for the full list of things to know.
