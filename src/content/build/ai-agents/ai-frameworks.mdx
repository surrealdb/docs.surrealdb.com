SurrealDB integrates with widely used AI and machine learning frameworks so you can use a single database for features, training or evaluation pipelines, and agent orchestration. Each integration page explains setup, typical patterns, and links to upstream documentation where helpful.

Available integrations:

- [Agno](/docs/build/integrations/ai-frameworks/agno)
- [CAMEL](/docs/build/integrations/ai-frameworks/camel)
- [CrewAI](/docs/build/integrations/ai-frameworks/crewai)
- [Dagster](/docs/build/integrations/ai-frameworks/dagster)
- [DeepEval](/docs/build/integrations/ai-frameworks/deepeval)
- [Dynamiq](/docs/build/integrations/ai-frameworks/dynamiq)
- [Feast](/docs/build/integrations/ai-frameworks/feast)
- [Google Agent Development Kit](/docs/build/integrations/ai-frameworks/google-agent)
- [LangChain](/docs/build/integrations/ai-frameworks/langchain)
- [LlamaIndex](/docs/build/integrations/ai-frameworks/llamaindex)
- [PydanticAI](/docs/build/integrations/ai-frameworks/pydantic-ai)
- [smolagents](/docs/build/integrations/ai-frameworks/smolagents)

For a comparative table and links to deeper guides, start from the [AI frameworks overview](/docs/build/integrations/ai-frameworks/overview).
