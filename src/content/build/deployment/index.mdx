# Deployment

SurrealDB can be deployed as a managed service via SurrealDB Cloud or self-hosted using Docker, Kubernetes or the standalone binary. Each option pairs with the same built-in **observability surface** — metrics, traces, audit logs and slow-query logs — so production deployments share a single operator runbook regardless of where the database runs.
