This section details the ways you can deploy SurrealDB, either via SurrealDB Cloud or a self-hosted instance using solutions like Docker or Kubernetes.

Also see the [Observability section](/docs/build/deployment/observability/overview) for observability tools such as **Prometheus (pull)** and **OTLP (push)** that are often indispensable when deploying a project in production.
