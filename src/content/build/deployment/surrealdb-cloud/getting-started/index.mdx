# Get started with SurrealDB Cloud

To get started, sign up for an account and create an instance. Once your instance is running, you can connect to it [via Surrealist](/docs/build/deployment/surrealdb-cloud/connecting/via-surrealist), [the CLI](/docs/build/deployment/surrealdb-cloud/connecting/via-cli), or [any of the supported SDKs](/docs/build/deployment/surrealdb-cloud/connecting/via-sdk).
