# Installation

Install the SDK from [pkg.go.dev](https://pkg.go.dev/github.com/surrealdb/surrealdb.go) using `go get`:

```bash
go get github.com/surrealdb/surrealdb.go
```

Then import the SDK and its models package in your Go files:

```go
import (
	surrealdb "github.com/surrealdb/surrealdb.go"
	"github.com/surrealdb/surrealdb.go/pkg/models"
)
```

The `surrealdb` package contains the client, query functions, and authentication methods. The `models` package contains value types such as [`RecordID`](/docs/sdk/golang/api/values/record-id), [`Table`](/docs/sdk/golang/api/values/table), and [`UUID`](/docs/sdk/golang/api/values/uuid).

## Requirements

- Go `1.23` or later
- SurrealDB `v2.x` or `v3.x`
