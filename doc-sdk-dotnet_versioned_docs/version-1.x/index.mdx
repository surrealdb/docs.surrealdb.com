# Overview

The SurrealDB SDK for .NET enables simple and advanced querying of a remote database from client or server-side code. Remote connections automatically reconnect when the connection is terminated.

To contribute to the SDK code, submit an Issue or Pull Request [here](https://github.com/surrealdb/surrealdb.net).

## Example projects 

You can find example repositories that demonstrate how to integrate SurrealDB in a number of different environments:

- [Console application](https://github.com/surrealdb/surrealdb.net/tree/main/SurrealDb.Examples.Console)
- [ASP.NET web API using classic Controllers](https://github.com/surrealdb/surrealdb.net/tree/main/SurrealDb.Examples.WeatherApi)
- [ASP.NET web API using Minimal APIs](https://github.com/surrealdb/surrealdb.net/tree/main/SurrealDb.Examples.MinimalApis)
- [ASP.NET web API with native AOT](https://github.com/surrealdb/surrealdb.net/blob/main/SurrealDb.Examples.TodoApi.Aot/Program.cs)
- [Blazor Server web application](https://github.com/surrealdb/surrealdb.net/tree/main/SurrealDb.Examples.Blazor.Server)
